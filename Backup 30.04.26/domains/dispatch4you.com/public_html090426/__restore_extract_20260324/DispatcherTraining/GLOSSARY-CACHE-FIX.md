
## 🐛 ЕСЛИ ВСЁ ЕЩЁ БОЛЬШИЕ КНОПКИ

### Вариант 1: Жёсткая очистка кэша
1. Откройте DevTools (F12)
2. Правый клик на кнопке обновления
3. Выберите "Очистить кэш и жёсткое обновление"

### Вариант 2: Очистка кэша браузера
1. Chrome/Edge: Settings → Privacy → Clear browsing data
2. Firefox: Settings → Privacy → Clear Data
3. Выберите "Cached images and files"
4. Нажмите "Clear data"

### Вариант 3: Другой браузер
Попробуйте открыть в другом браузере для проверки.
