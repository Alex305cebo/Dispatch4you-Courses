# ✅ БЫСТРЫЙ ЧЕК-ЛИСТ: Исправление меню

## 🎯 ЦЕЛЬ
Убрать наложение мобильного меню на десктопное меню

---

## 📋 ШАГ ЗА ШАГОМ

### ☐ 1. Войдите в Hostinger
- [ ] Откройте https://hpanel.hostinger.com
- [ ] Войдите в аккаунт
- [ ] Откройте File Manager

### ☐ 2. Загрузите shared-nav.css (ГЛАВНЫЙ ФАЙЛ!)
- [ ] Перейдите в `public_html`
- [ ] Удалите старый `shared-nav.css`
- [ ] Загрузите новый `shared-nav.css` (версия v3.0)
- [ ] Проверьте, что файл загрузился

### ☐ 3. Загрузите index.html
- [ ] Удалите старый `index.html`
- [ ] Загрузите новый `index.html`
- [ ] Проверьте, что файл загрузился

### ☐ 4. Загрузите pages/role.html
- [ ] Откройте папку `pages`
- [ ] Удалите старый `role.html`
- [ ] Загрузите новый `role.html`
- [ ] Проверьте, что файл загрузился

### ☐ 5. Очистите кэш
- [ ] Очистите кэш браузера (Ctrl + Shift + Delete)
- [ ] Сделайте жёсткую перезагрузку (Ctrl + Shift + R)

### ☐ 6. Проверьте результат
- [ ] Откройте сайт на десктопе (ширина > 1024px)
- [ ] Убедитесь, что мобильное меню НЕ показывается
- [ ] Откройте сайт на мобильном (ширина < 768px)
- [ ] Убедитесь, что мобильное меню работает
- [ ] Проверьте в Chrome
- [ ] Проверьте в Firefox
- [ ] Проверьте в Safari (если есть)

---

## 🔍 БЫСТРАЯ ПРОВЕРКА ВЕРСИИ

Откройте в браузере:
```
https://dispatch4you.com/shared-nav.css
```

Первая строка должна быть:
```
NAVBAR — shared-nav.css v3.0 COMPLETE REWRITE
```

Если видите старую версию → очистите кэш!

---

## ❌ ЕСЛИ НЕ РАБОТАЕТ

1. Проверьте, что файл `shared-nav.css` на сервере имеет версию v3.0
2. Очистите кэш браузера ещё раз
3. Попробуйте открыть сайт в режиме инкогнито
4. Проверьте консоль браузера (F12) на ошибки
