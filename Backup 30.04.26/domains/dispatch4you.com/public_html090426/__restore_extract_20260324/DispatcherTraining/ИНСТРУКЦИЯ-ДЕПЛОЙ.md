```bash

git add .

# 3. Запушить в оба репозитория
```

## 🚨 Частые ошибки

### Ошибка 1: Изменения не появляются на сайте
**Причина:** Запушили только в `origin`, забыли про `site`

**Решение:**
```bash
git push site main
```

### Ошибка 2: Кэш браузера
**Причина:** Браузер показывает старую версию

**Решение:**
```
Ctrl + Shift + Delete → Очистить кэш
Ctrl + Shift + R → Жесткая перезагрузка
```

### Ошибка 3: Hostinger не обновляется
**Причина:** Нужно подождать или перезапустить деплой

**Решение:**
```
1. Зайти в Hostinger панель
2. Git → Manage Repositories
3. Нажать "Deploy" вручную
```
