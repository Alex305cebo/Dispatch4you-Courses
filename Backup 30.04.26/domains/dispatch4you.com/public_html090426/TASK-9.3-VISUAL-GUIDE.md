## Дополнительные эффекты

### 1. Toast уведомление (от xp-system.js)
После начисления XP пользователь видит toast в правом нижнем углу:

```
┌────────────────────────┐
│  ⚡  +50 XP            │
│  📞 Сессия с брокером  │
│     завершена          │
└────────────────────────┘
```

Если есть бонус, показывается дополнительный toast:

```
┌────────────────────────┐
│  ⚡  +25 XP            │
│  ⭐ Высокий балл (8+)  │
└────────────────────────┘
```

### 2. Обновление навигации
XP в навигационной панели обновляется автоматически:

**До:**
```
👤 Иван Петров  |  ⚡ 450 XP  |  🎓 Диспетчер
```

**После (с бонусом 25 XP):**
```
👤 Иван Петров  |  ⚡ 525 XP  |  🎓 Диспетчер
```

## Граничные случаи

### Граница 8.0 (включительно)
```
avgScore = 8.0
→ Начислено 75 XP (50 базовых + 25 бонус: высокий балл (8+))
```

### Граница 9.0 (включительно)
```
avgScore = 9.0
→ Начислено 100 XP (50 базовых + 50 бонус: отличный балл (9+))
```

### Чуть ниже границы
```
avgScore = 7.9
→ Начислено 50 XP (без бонуса)

avgScore = 8.9
→ Начислено 75 XP (50 базовых + 25 бонус: высокий балл (8+))
```

## Цветовая схема

### XP Highlight (всегда зелёный)
- **Фон**: `rgba(16, 185, 129, 0.08)` (светло-зелёный)
- **Граница**: `rgba(16, 185, 129, 0.2)` (зелёный)
- **Иконка**: ⚡ (молния)
- **Тип**: `success`

### Другие Highlights
- **Success** (✅): зелёный фон
- **Improvement** (💡): оранжевый фон

---

## Порядок отображения

1. **Первым всегда**: XP highlight (⚡)
2. **Затем**: Success highlights (✅)
3. **В конце**: Improvement highlights (💡)

Пример:
```
✨ Ключевые моменты
┌─────────────────────────────────────────┐
│ ⚡ Начислено 75 XP (50 + 25 бонус)      │  ← XP (всегда первый)
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│ ✅ Правильно использовал "rate con"     │  ← Success
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│ ✅ Уверенно вёл переговоры              │  ← Success
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│ 💡 Можно было уточнить detention policy │  ← Improvement
└─────────────────────────────────────────┘
```

## Анимация

1. **Модальное окно**: fade in + scale (0.3s)
2. **Toast уведомления**: slide in справа (0.4s)
3. **Обновление XP в навигации**: плавное изменение числа

---

## Доступность

- ✅ Иконки имеют текстовое описание
- ✅ Цвета соответствуют WCAG AA
- ✅ Текст читаем на всех фонах
- ✅ Кнопки имеют достаточный размер для клика
- ✅ Модальное окно можно закрыть клавишей Escape (если реализовано)

---

## Примечания для разработчиков

1. XP highlight всегда добавляется первым в массив `highlightsHTML`
2. Проверка `if (bonus > 0)` определяет, показывать ли детали бонуса
3. Функция `esc()` используется для безопасного отображения текста
4. Модальное окно показывается через класс `.show`
5. Навигация обновляется автоматически через событие `xpUpdated`
