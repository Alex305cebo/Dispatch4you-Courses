## Тестовые сценарии

### Сценарий 1: Полный цикл сессии с голосовым вводом

**Шаги:**
1. Открыть `pages/ai-broker-chat.html`
2. Выбрать сценарий (например, "Свободный")
3. Нажать "Начать звонок" на prep screen
4. Нажать кнопку микрофона
5. Говорить на английском
6. Наблюдать interim transcript (серый текст)
7. Дождаться final transcript (белый текст)
8. Автоматическая отправка через 1.5 сек
9. AI отвечает (без fallback)
10. Повторить 3-4 раза
11. Нажать "Завершить сессию"
12. Проверить модальное окно с метриками
13. Проверить начисление XP

**Ожидаемый результат:**
- ✅ Interim transcript показывается в реальном времени
- ✅ Final transcript отправляется автоматически
- ✅ AI отвечает через Groq API
- ✅ Модальное окно показывает метрики (1-10)
- ✅ XP начисляется (50 + бонус если avgScore >= 8)
- ✅ Данные сохраняются в Firestore

### Сценарий 2: Проверка истории сессий

**Шаги:**
1. Завершить несколько сессий (см. Сценарий 1)
2. Нажать кнопку "📜" (История)
3. Проверить список сессий
4. Кликнуть на сессию
5. Проверить детали: сообщения, метрики, обратная связь

**Ожидаемый результат:**
- ✅ Список сессий загружается из Firestore
- ✅ Показывается дата, сценарий, балл, XP
- ✅ Детали сессии показывают все сообщения с timestamps
- ✅ Метрики отображаются корректно

### Сценарий 3: Обработка ошибок Speech Recognition

**Шаги:**

### Сценарий 5: XP начисление по баллам

**Шаги:**
1. Завершить сессию с avgScore < 8
2. Проверить XP: должно быть 50
3. Завершить сессию с avgScore >= 8 и < 9
4. Проверить XP: должно быть 75 (50 + 25)
5. Завершить сессию с avgScore >= 9
6. Проверить XP: должно быть 100 (50 + 50)

**Ожидаемый результат:**
- ✅ Базовый XP: 50
- ✅ Бонус за высокий балл (8+): +25
- ✅ Бонус за отличный балл (9+): +50
- ✅ XP отображается в модальном окне
- ✅ XP обновляется в навигации
