## Технические детали

### Структура xpAwarded
```javascript
{
  total: 50,        // Общая сумма XP
  base: 50,         // Базовые XP (всегда 50)
  bonus: 0,         // Бонусные XP (0, 25 или 50)
  bonusReason: ''   // Причина бонуса (пустая строка или текст)
}
```

### Логика отображения
```javascript
if (metrics.xpAwarded) {
  const { total, base, bonus, bonusReason } = metrics.xpAwarded;
  let xpMessage = `Начислено ${total} XP`;
  if (bonus > 0) {
    xpMessage += ` (${base} базовых + ${bonus} бонус: ${bonusReason})`;
  }
  // Отображается как первый highlight с иконкой ⚡
}
```

### Интеграция с xp-system.js
- XP начисляются через `awardXP()` функцию из `xp-system.js`
- Используются действия: `BROKER_SESSION_COMPLETE`, `BROKER_HIGH_SCORE`, `BROKER_PERFECT_SCORE`
- Навигация обновляется автоматически через событие `xpUpdated` в xp-system.js
- Показывается toast уведомление о начислении XP
