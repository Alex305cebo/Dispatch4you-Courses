### 2. Автоматическое раскрытие секции "Курс обучения" в мобильном меню
**Проблема:** При открытии бургер-меню все секции были закрыты

**Решение:**
- Файл: `nav-loader.js` (обновлен до v7.1)
- Добавлено: Автоматическое раскрытие первой секции при открытии мобильного меню
- Код:
```javascript
function openMenu() {
    var menu = document.getElementById('mobMenu');
    var overlay = document.getElementById('mobOverlay');
    if (menu) menu.classList.add('active');
    if (overlay) overlay.classList.add('active');
    document.body.style.overflow = 'hidden';
    
    // Автоматически раскрываем первую секцию "Курс обучения"
    setTimeout(function() {
        var firstAccordion = document.querySelector('.mob-acc');
        if (firstAccordion && !firstAccordion.classList.contains('open')) {
            firstAccordion.classList.add('open');
        }
    }, 100);
}
```
