🚛 Основные типы траков

В грузоперевозках США используются различные типы тягачей и прицепов, каждый из которых предназначен для определенных видов грузов.

🚚 Dry Van

Стандартный закрытый прицеп 53 фута. Самый распространенный тип для перевозки упакованных товаров.

❄️ Reefer

Рефрижераторный прицеп с контролем температуры. Для перевозки скоропортящихся продуктов.

🏗️ Flatbed

Открытая платформа для негабаритных грузов, строительных материалов и оборудования.

🚛 Step Deck

Платформа с пониженной высотой для высоких грузов, не требующих специальных разрешений.

🔧 Power Only

Только тягач без прицепа. Водитель подцепляет прицеп заказчика на месте.

📦 Box Truck

Цельный грузовик с кузовом. Используется для местных доставок и небольших грузов.

📏 Размеры и характеристики

  • Стандартная длина прицепа: 53 фута (16.15 м)
  • Максимальный вес: 80,000 lbs (36,287 кг)
  • Высота стандартного прицепа: 13'6" (4.11 м)
  • Ширина: 8'6" (2.59 м)
  • Вместимость Dry Van: 26 паллет стандартного размера

⚙️ Специальное оборудование

  • Conestoga - прицеп с раздвижным тентом
  • Double Drop - для сверхтяжелых грузов
  • Lowboy - для строительной техники
  • Tanker - для жидких грузов
  • Auto Hauler - для перевозки автомобилей
  • Hotshot - пикап с прицепом для срочных доставок
