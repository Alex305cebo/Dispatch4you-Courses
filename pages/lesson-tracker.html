• Может ли диспетчер работать без договора с carrier?
💡 Аналогии для объяснения
Диспетчер = Риелтор: не владеет квартирой, но помогает продать и берёт комиссию

Broker = Агент по найму: берёт заказ от компании (shipper) и находит исполнителя (carrier)

Rate Con = Договор аренды: юридически обязывающий документ, нельзя нарушить без последствий

DAT = Авито для грузов: биржа где брокеры публикуют грузы, а диспетчеры их находят

Q4 = Чёрная пятница: пик спроса, все хотят доставить товары к Рождеству
⚠️ Типичные ошибки студентов
Путают диспетчера с брокером — напомни: брокер имеет лицензию MC и работает между shipper и carrier. Диспетчер — агент carrier без лицензии

Думают что нужна лицензия — нет. Диспетчер = independent contractor по договору

Завышают ожидания по доходу — первый месяц $300–800, не $5,000

Не понимают RPM — объясни: Rate Per Mile = ставка / мили. Минимум $1.80/mile для Dry Van

Боятся звонить брокерам — практикуй скрипт до автоматизма на каждом уроке
🚀 Факты для мотивации студента
• В США не хватает 80,000 водителей грузовиков — рынок диспетчеров растёт
• Средний диспетчер в США зарабатывает $45,000–$65,000 в год
• Топовые диспетчеры с 5–10 траками зарабатывают $100,000+ в год
• Профессия полностью удалённая — работают из 50+ стран мира
• Для старта нужно $200–500 (подписка на Load Board + телефон)
• Первый груз можно забукировать уже через 2–3 недели обучения
• Amazon, Walmart, Target — крупнейшие shippers, тысячи грузов ежедневно
• Рынок e-commerce растёт на 15–20% в год → больше грузов → выше ставки
🎓 Приёмы эффективного обучения
Метод Сократа: не давай ответ сразу — задавай наводящие вопросы пока студент сам не придёт к выводу

Калькулятор мечты: спроси желаемый доход → посчитайте вместе сколько траков нужно

Ролевая игра: ты — брокер, студент — диспетчер. Практикуй звонок на каждом уроке

Якорение: в конце урока — одно конкретное действие которое студент сделает сегодня

Визуализация: «Представь что ты едешь по шоссе и считаешь траки — каждый потенциальный клиент»
