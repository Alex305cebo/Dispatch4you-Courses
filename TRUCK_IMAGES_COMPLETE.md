## 🎨 Анимация траков

### Idle (стоящие)
- Легкое покачивание (1.2 сек цикл)
- Показывает картинку трака

### Driving / Loaded (едущие)
- Движение вверх-вниз (0.6 сек цикл)
- Показывает картинку трака

### At Pickup (на погрузке)
- Эмодзи 📦
- Пульсация (0.7 сек цикл)

### At Delivery (на разгрузке)
- Эмодзи 🏁
- Пульсация (0.7 сек цикл)

### Breakdown (поломка)
- Эмодзи 🔧
- Вращение влево-вправо

### Waiting (detention)
- Эмодзи ⏳
- Без анимации

### Night Stop (ночёвка)
- Эмодзи 🌙
- Без анимации

## 🔧 Технические детали

### Код изменений

**MapView.tsx:**
- Добавлен маппинг `truckImageId` → картинка
- Логика отображения: картинка для обычных статусов, эмодзи для специальных
- Анимация в зависимости от статуса

**gameStore.ts:**
- Добавлено поле `truckImageId` в тип `Truck`
- Стартовые траки получают разные ID (1-10)
- При покупке трака сохраняется `lotId` как `truckImageId`

**TruckShopModal.tsx:**
- Использует WebP формат с функцией `getTruckImage()`
- Fallback на PNG для совместимости

## 🚀 Деплой

### Локальное тестирование
```bash
cd adventure
npm start
# Откройте http://localhost:8082
```

### Production сборка
```bash
cd adventure
npm run build:web
# Файлы в dist/ готовы к деплою
```

### Копирование на сервер
```bash
# Локально
Copy-Item "adventure/dist/*" "game/" -Recurse -Force

# Или через Git
git add .
git commit -m "Add truck images on map"
git push origin main
```
