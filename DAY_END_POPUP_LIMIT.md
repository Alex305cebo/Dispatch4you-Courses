# Ограничение показа экрана "Рабочий день завершён!"

## Проблема
Экран "Рабочий день завершён!" показывался каждый раз при завершении игрового дня, что могло раздражать при частой игре.

## Решение
Добавлена проверка через `localStorage` — экран показывается **не чаще одного раза в реальный день**.

## Как работает

### 1. Проверка при показе
```typescript
const shouldShowToday = () => {
  const lastShown = localStorage.getItem('dayEndPopup_lastShown');
  const today = new Date().toDateString();
  
  if (lastShown === today) {
    return false; // Уже показывали сегодня
  }
  
  // Сохраняем текущую дату
  localStorage.setItem('dayEndPopup_lastShown', today);
  return true;
};
```

### 2. Автоматический пропуск
Если экран уже показывался сегодня:
- Попап не показывается
- Игра автоматически переходит в режим `playing`
- Статистика траков обновляется (yesterdayMiles)

### 3. Сброс в полночь
Проверка использует `new Date().toDateString()`, который возвращает дату в формате:
```
"Mon Jan 01 2024"
```

Это означает что:
- ✅ В 00:00 следующего дня экран снова можно показать
- ✅ Работает в любом часовом поясе
- ✅ Не зависит от перезагрузки страницы

## Примеры использования

### Сценарий 1: Первый раз за день
```
Пользователь завершает игровой день → 
Проверка localStorage → Нет записи за сегодня →
Показываем экран "Рабочий день завершён!" →
Сохраняем дату в localStorage
```

### Сценарий 2: Повторное завершение дня
```
Пользователь завершает игровой день снова →
Проверка localStorage → Есть запись за сегодня →
Пропускаем экран →
Автоматически переходим к playing
```

### Сценарий 3: Следующий день
```
Наступает новый реальный день (00:00) →
Пользователь завершает игровой день →
Проверка localStorage → Запись за вчера (не совпадает) →
Показываем экран →
Обновляем дату в localStorage
```

## Технические детали

### localStorage ключ
```
dayEndPopup_lastShown
```

### Формат значения
```
"Mon Jan 01 2024"
```

### Очистка данных
Для тестирования или сброса можно выполнить в консоли:
```javascript
localStorage.removeItem('dayEndPopup_lastShown');
```

Или очистить всё:
```javascript
localStorage.clear();
```

## Преимущества

✅ **Не раздражает пользователя** — показывается максимум раз в день
✅ **Сохраняет важность** — первый раз за день всё равно показывается
✅ **Простая реализация** — всего несколько строк кода
✅ **Надёжно** — работает даже после перезагрузки страницы
✅ **Автоматический сброс** — в полночь счётчик обнуляется
