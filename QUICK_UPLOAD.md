# ⚡ Быстрая загрузка обновлений плеера

## 🎯 Что делать (3 простых шага)

### 1️⃣ Зайти в File Manager
```
https://hpanel.hostinger.com
→ Выбрать dispatch4you.com
→ Нажать "File Manager"
→ Открыть папку "public_html"
```

### 2️⃣ Загрузить 3 файла

Удалите старые версии и загрузите новые:

| Файл | Где находится | Куда загрузить |
|------|---------------|----------------|
| `lesson-audio.css` | Корень проекта | `public_html/` |
| `lesson-audio.js` | Корень проекта | `public_html/` |
| `nav.html` | Корень проекта | `public_html/` |

**Как загрузить:**
1. Найти файл в File Manager
2. Правая кнопка → Delete (удалить старый)
3. Кнопка "Upload Files" → выбрать новый файл
4. Дождаться загрузки ✅

### 3️⃣ Очистить кэш

**В браузере:**
- `Ctrl + Shift + Delete` → очистить кэш
- `Ctrl + F5` → жесткая перезагрузка

**На сайте:**
- Открыть `/pages/documentation.html`
- Проверить что плеер сворачивается/разворачивается

1. Проверьте консоль браузера (F12)
2. Убедитесь что файлы в правильной папке
