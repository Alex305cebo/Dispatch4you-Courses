# 🚀 БЫСТРЫЙ СТАРТ

## ✅ Что сделано

1. **Новое главное меню 2026** — современный игровой интерфейс

## 🎮 Как запустить

### 1. Установка зависимостей
```bash
cd adventure
npm install
```

### 2. Запуск dev сервера
```bash
npm start
```

### 3. Открыть в браузере
```
http://localhost:8081
```

## 🎯 Основные фичи

### 1. Гибридные сохранения
- Локально (localStorage) — быстро
- Облако (Firebase) — синхронизация
- Автоматический выбор лучшего варианта

### 2. История сохранений
- Последние 5 сохранений в облаке
- Загрузка любого сохранения
- Информация о каждом (время, баланс, траки)

### 3. Оптимизация производительности
- Авто-определение устройства
- 3 режима качества (Low/Medium/High)
- Динамический tick interval (250ms-1000ms)

### 4. Современный UI
- Тёмная тема с градиентами
- Анимированный фон
- Светящиеся эффекты
- Плавные переходы

## 🐛 Если что-то не работает

### Ошибка: "Unable to resolve firebase/app"
```bash
npm install firebase
```

### Ошибка: "Cannot find module menu"
```bash
# Проверь что файл существует
ls app/menu.tsx

# Перезапусти сервер
npm start
```

### Белый экран
1. Открой консоль браузера (F12)
2. Посмотри ошибки
3. Перезагрузи страницу (Ctrl+R)

### Не загружается сохранение
1. Проверь авторизацию
2. Открой "Мои сохранения"
3. Выбери нужное сохранение вручную
