# 🗺️ Google Maps для Dispatch Office

## 🎉 Революция в отслеживании траков!

Старая схематичная карта **amCharts** заменена на **Google Maps** с реальными дорогами, спутниковой картой и пробками в реальном времени!

---

## 🚀 Быстрый старт (3 минуты)

### 1. Получите API ключ

1. Откройте https://console.cloud.google.com/
2. Создайте проект
3. Включите **Maps JavaScript API** и **Directions API**
4. Создайте API ключ
5. Скопируйте ключ (выглядит как `AIzaSyC...`)

### 2. Добавьте ключ в проект

Создайте файл `.env` в этой папке:

```bash
EXPO_PUBLIC_GOOGLE_MAPS_API_KEY=AIzaSyC...ваш_ключ
```

### 3. Замените карту

В файле `app/game.tsx` (строка 24):

**Было:**
```typescript
import MapView from '../components/MapView';
```

**Стало:**
```typescript
import MapView from '../components/GoogleMapView';
```

### 4. Запустите проект

```bash
npm start
```

**Готово!** Откройте браузер и проверьте карту.

## 🐛 Проблемы?

### Карта не загружается

1. Проверьте API ключ в `.env`
2. Откройте консоль браузера (F12)
3. Проверьте что **Maps JavaScript API** включён в Google Cloud Console

### Маршруты не рисуются

1. Проверьте что **Directions API** включён
2. Проверьте квоту в Google Cloud Console
3. Откройте консоль и проверьте ошибки

### "RefererNotAllowedMapError"

1. Откройте Google Cloud Console → Credentials
2. Нажмите на ваш API ключ
3. Добавьте домен в **HTTP referrers**:
   ```
   localhost:8081/*
   yourdomain.com/*
   ```
