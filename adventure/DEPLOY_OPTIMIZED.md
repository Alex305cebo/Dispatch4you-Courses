## 🚀 Как задеплоить

### Вариант 1: Быстрый деплой (рекомендуется)

```bash
cd adventure
npm run build:web
```

Затем скопируйте содержимое папки `dist/` на сервер в `/game/`

### Вариант 2: Локальная сборка

```bash
cd adventure
npm run build:local
```

### Проверка после деплоя

1. Откройте игру в браузере
2. Откройте DevTools (F12) → Network
3. Найдите запросы к картинкам траков
4. Убедитесь что загружаются `.webp` файлы (14-27 KB)

## 📊 Ожидаемые результаты

- Загрузка всех 11 траков: ~1.3 MB

### После оптимизации
- Загрузка всех 11 траков: ~209 KB (WebP)

## 🔧 Если нужно добавить новые траки

1. Добавьте PNG файлы в `Truck Pic/`
2. Запустите оптимизацию:
   ```bash
   node scripts/optimize-truck-images.js
   ```
3. Скопируйте оптимизированные файлы:
   ```bash
   Copy-Item "Truck Pic Optimized/*" "Truck Pic/" -Force
   ```
4. Пересоберите проект:
   ```bash
   npm run build:web
   ```

## ⚠️ Важно

- **НЕ удаляйте** папку `Truck Pic Backup/` — там оригиналы
- **НЕ коммитьте** папку `Truck Pic Optimized/` — она временная
- **Всегда запускайте** `npm run build:web` перед деплоем
