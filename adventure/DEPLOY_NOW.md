## 🚀 Деплой на Hostinger

### Вариант 1: Через File Manager (рекомендуется)

1. **Открой hPanel**
   - Зайди на https://hpanel.hostinger.com
   - Выбери свой сайт

2. **Открой File Manager**
   - Files → File Manager
   - Перейди в `/public_html/game/`

3. **Удали старые файлы**
   - Выдели все файлы в `/public_html/game/`
   - Нажми Delete
   - **НЕ удаляй** `.htaccess` если он там есть

4. **Загрузи новые файлы**
   - Нажми Upload
   - Выбери **ВСЁ** из папки `C:\DispatcherTraining\adventure\dist\`
   - Дождись завершения загрузки

5. **Проверь структуру**
   ```
   /public_html/game/
   ├── index.html
   ├── favicon.ico
   ├── metadata.json
   ├── _expo/
   │   └── static/
   │       ├── css/
   │       └── js/
   └── assets/
       ├── TruckPic/
       └── avatars/
   ```

### Вариант 2: Через FTP

1. **Подключись к FTP**
   - Host: `ftp.dispatch4you.com` (или твой домен)
   - Username: из hPanel
   - Password: из hPanel
   - Port: 21

2. **Перейди в папку**
   - Открой `/public_html/game/`

3. **Удали старые файлы**
   - Выдели все файлы

4. **Загрузи новые**
   - Перетащи всё из `dist/` в `/public_html/game/`

## 🐛 Если что-то не работает

### Ошибка: "initGoogleMap is not a function"

**Решение:**
1. Полностью перезагрузи страницу: `Ctrl + Shift + R`
2. Очисти кэш браузера
3. Попробуй в режиме инкогнито

### Ошибка: "Failed to load resource"

**Причина:** Не все файлы загрузились

**Решение:**
1. Проверь что папка `_expo/` загружена полностью
2. Проверь что папка `assets/` загружена полностью
3. Перезагрузи недостающие файлы

### Ошибка: Белый экран

**Причина:** Неправильные пути в `index.html`

**Решение:**
1. Открой `/public_html/game/index.html`
2. Проверь что пути начинаются с `/game/`:
   ```html
   href="/game/_expo/static/css/..."
   src="/game/_expo/static/js/..."
   ```
3. Если нет — запусти сборку заново: `npm run build:web`

### Старая версия всё ещё показывается

**Причина:** Кэш браузера или CDN

**Решение:**
1. Очисти кэш браузера: `Ctrl + Shift + R`
2. Попробуй в режиме инкогнито
3. Подожди 5-10 минут (кэш CDN обновится)
