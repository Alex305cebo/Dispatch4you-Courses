# Интерактивная карта — готово! ✅

## Что добавлено

### 1. Подсветка штатов при наведении
- Наведи курсор на любой штат — он станет светлее (#2d6a4f вместо тёмно-зелёного)
- Граница штата станет ярко-зелёной (#4ade80) и толще
- Появится тень вокруг штата

### 2. Tooltip с названием штата
- При наведении на штат показывается его название
- Tooltip появляется рядом с курсором
- Траки имеют приоритет — если навести на трак, показывается информация о траке

### 3. Курсор меняется
- При наведении на штат курсор становится pointer (рука)
- На пустом месте — обычный курсор

## Как проверить

1. Открой игру в браузере: http://localhost:8081
2. Сделай жёсткую перезагрузку: **Ctrl+Shift+R** (Windows) или **Cmd+Shift+R** (Mac)
3. Если не помогло:
   - Открой DevTools (F12)
   - Вкладка Application → Clear storage
   - Нажми "Clear site data"
   - Перезагрузи страницу

4. Наведи курсор на любой штат на карте — он должен подсветиться

## Технические детали

### Как это работает
- `hoveredState` — state который хранит название штата под курсором
- `hoveredStateRef` — ref для доступа к hoveredState внутри animation loop
- `isPointInPath()` — Canvas API для проверки попадания курсора в полигон штата
- Перерисовка карты происходит каждый кадр (requestAnimationFrame)

### Производительность
- Проверка штатов происходит только при движении мыши
- Рендер оптимизирован — используется один Canvas context
- Нет лишних перерисовок — только при изменении hoveredState
