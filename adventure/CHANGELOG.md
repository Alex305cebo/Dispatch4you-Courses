# Changelog - Office 4 Dispatch

## [2.2.0-beta.1] - 2026-04-18

### 🎯 Новое
- ✨ Добавлена система визуальной обратной связи для всех интерактивных элементов
- ✨ Ripple эффект при клике на кнопки и карточки
- ✨ Увеличенные крестики закрытия (48×48px desktop, 56×56px mobile)
- ✨ Вибрация на мобильных устройствах при нажатии
- ✨ Улучшенные hover эффекты для всех кнопок

### 🔧 Улучшения
- 🎨 Все кнопки теперь имеют плавные анимации
- 🎨 Карточки траков поднимаются при наведении
- 🎨 Крестики закрытия модальных окон увеличены для удобства на мобильных
- 📱 Оптимизация для touch устройств
- ⚡ Автоматическое применение эффектов ко всем элементам

### 🐛 Исправления
- 🔨 Исправлена проблема с непонятным состоянием кнопок при нажатии
- 🔨 Улучшена читаемость интерфейса на мобильных

### 📦 Технические изменения
- Добавлены файлы: `interactive-feedback.css`, `interactive-feedback.js`
