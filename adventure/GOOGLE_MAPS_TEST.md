# 🧪 Тестирование Google Maps API

## Шаг 1: Тест API ключа

Откройте в браузере:
```
http://localhost:8081/test-google-maps.html
```

### Что должно произойти:

✅ **Успех**: Вы увидите карту США с маркером в центре и зелёное сообщение "Google Maps API работает отлично!"

❌ **Ошибка**: Красное сообщение с описанием проблемы

### Возможные ошибки:

1. **RefererNotAllowedMapError** - Домен не разрешён в ограничениях API ключа
   - Решение: Добавьте `localhost:8081/*` в HTTP referrers в Google Cloud Console

2. **ApiNotActivatedMapError** - API не включен
   - Решение: Включите Maps JavaScript API в Google Cloud Console

3. **InvalidKeyMapError** - Неверный API ключ
   - Решение: Проверьте API ключ в файле `.env`

4. **Карта не загружается** - Проблема с сетью или скриптом
   - Решение: Откройте консоль браузера (F12) и проверьте ошибки

## Шаг 2: Тест в игре

После успешного теста откройте игру:
```
http://localhost:8081
```

1. Запустите новую сессию или продолжите существующую
2. Перейдите на вкладку "Карта" (кнопка с картой внизу)
3. Вы должны увидеть Google Maps с траками на карте

### Что проверить:

- ✅ Карта загружается без ошибок
- ✅ Траки отображаются как цветные точки
- ✅ Маршруты рисуются линиями между городами
- ✅ Клик по траку показывает информацию
- ✅ Легенда статусов отображается в правом верхнем углу

## Отладка

### Проблема: "Ошибка появляется много раз"

**Причина**: Компонент пытается загрузить API несколько раз

**Решение**: Исправлено в новой версии `GoogleMapView.tsx` - теперь проверяется наличие скрипта в DOM

### Проблема: API ключ не читается из .env

**Причина**: Expo не подхватывает переменные окружения

**Решение**: 
1. Остановите сервер (Ctrl+C)
2. Очистите кеш: `npx expo start --clear`
3. Перезапустите: `npx expo start --web --port 8081`

### Проблема: Ограничения API ключа

**Проверьте в Google Cloud Console:**
- Application restrictions: HTTP referrers
- Website restrictions должны включать:
  - `localhost:8081/*`
  - `localhost:19006/*`
  - `dispatch4you.com/*`
  - `www.dispatch4you.com/*`
