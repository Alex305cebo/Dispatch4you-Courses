# Чеклист — Система выбора трака

## Перед запуском

- [ ] Установлены зависимости (`npm install`)
- [ ] Metro bundler запущен (`npm start`)
- [ ] Игра открыта в браузере/эмуляторе
- [ ] Создана новая сессия с 3-5 траками

## Базовый функционал

### Автоматический выбор трака

- [ ] После успешных переговоров показывается карточка трака
- [ ] Выбран трак со статусом `idle`, `at_delivery` или `at_pickup`
- [ ] Показано название трака и имя водителя
- [ ] Показано текущее местоположение
- [ ] Показаны HOS часы (например "11.0ч")
- [ ] Показано настроение (например "70%")
- [ ] Если deadhead > 0 — показано расстояние (например "🛣️ Deadhead: 180 миль")

### Логика выбора

- [ ] Если несколько траков в одном городе → выбран с лучшим HOS
- [ ] Если траки в разных городах → выбран ближайший к погрузке
- [ ] Idle траки имеют приоритет над at_delivery/at_pickup
- [ ] Траки со статусом `driving`, `loaded`, `breakdown` не доступны

### Смена трака

- [ ] Если доступно 2+ трака — показана кнопка "Изменить"
- [ ] Клик на "Изменить" открывает список траков
- [ ] В списке показаны все доступные траки
- [ ] У каждого трака показан deadhead
- [ ] Выбранный трак отмечен галочкой ✓
- [ ] Клик на трак выбирает его и закрывает список
- [ ] Карточка обновляется с новым траком

### Подтверждение

- [ ] Кнопка "✅ Подтвердить" активна когда трак выбран
- [ ] Клик на "Подтвердить" назначает груз
- [ ] Окно переговоров закрывается
- [ ] Трак начинает движение к погрузке
- [ ] На карте видно движение трака
- [ ] В "My Loads" появился груз

### Отмена

- [ ] Кнопка "Отмена" всегда доступна
- [ ] Клик на "Отмена" закрывает окно
- [ ] Груз НЕ назначен
- [ ] Трак остался в прежнем статусе

## Prebook (резервирование)

### Трак на разгрузке

- [ ] Трак со статусом `at_delivery` доступен для выбора
- [ ] Показан статус "🔄 Заканчивает разгрузку — начнёт после"
- [ ] Можно подтвердить назначение
- [ ] После подтверждения груз назначен как "следующий"
- [ ] Трак продолжает разгрузку
- [ ] После разгрузки трак автоматически начинает движение к погрузке

### Трак на погрузке

- [ ] Трак со статусом `at_pickup` доступен для выбора
- [ ] Показан статус "🔄 Заканчивает погрузку — начнёт после"
- [ ] Можно подтвердить назначение
- [ ] После подтверждения груз назначен как "следующий"
- [ ] Трак продолжает погрузку
- [ ] После погрузки трак автоматически начинает движение к новой погрузке

## Граничные случаи

### Нет свободных траков

- [ ] Все траки заняты (status = `driving` или `loaded`)
- [ ] Кнопка "📞 Позвонить брокеру" неактивна
- [ ] Показано сообщение "🚫 Нет свободных траков"
- [ ] Нельзя начать переговоры

### Один свободный трак

- [ ] Автоматически выбран единственный доступный трак
- [ ] Кнопка "Изменить" НЕ показана
- [ ] Можно сразу подтвердить

### Все траки далеко

- [ ] Выбран ближайший трак
- [ ] Показан большой deadhead (например 1000+ миль)
- [ ] Можно изменить выбор
- [ ] Можно подтвердить

### Трак с низким HOS

- [ ] Трак с HOS < 3 часа доступен для выбора
- [ ] Показано предупреждение (если реализовано)
- [ ] Можно выбрать другой трак с лучшим HOS

## Расчёты

### Deadhead

- [ ] Deadhead = 0 для трака в том же городе что и погрузка
- [ ] Deadhead > 0 для трака в другом городе
- [ ] Deadhead рассчитан корректно (проверить на карте)
- [ ] Deadhead показан в милях

### Score

- [ ] Трак с меньшим deadhead имеет больший score
- [ ] Трак с большим HOS имеет больший score
- [ ] Idle трак имеет больший score чем at_delivery
- [ ] Score рассчитан корректно (проверить в консоли)
