## Технические детали

### Архитектура
- **SVG карта**: `@mirawision/usa-map-react` — рендерит штаты
- **Траки**: Отдельный SVG слой поверх карты
- **Hover**: Работает нативно через SVG events
- **Координаты**: Простая линейная проекция lat/lng → x/y

### Преимущества над Canvas
1. **Hover работает из коробки** — не нужна логика проверки координат
2. **CSS стили** — можно стилизовать через CSS
3. **Производительность** — браузер оптимизирует SVG
4. **Доступность** — работает с клавиатурой
5. **Легко кастомизировать** — каждый штат это path элемент

### Что удалено
- ❌ d3-geo (больше не нужен)
- ❌ topojson-client (больше не нужен)
- ❌ usStates.ts (больше не нужен)
- ❌ Canvas логика
- ❌ requestAnimationFrame loop
- ❌ Ручная проверка координат

## Кастомизация

### Изменить цвета штатов

В `MapView.tsx`:

```tsx
fill: hoveredState === state ? '#2d6a4f' : '#1a3a2a',
stroke: hoveredState === state ? '#4ade80' : '#2d6a4f',
```

### Изменить hover эффект

В `global.css`:

```css
.usa-map-react path:hover {
  filter: drop-shadow(0 0 8px rgba(45,106,79,0.6));
  cursor: pointer;
}
```

### Добавить клик по штату

В `MapView.tsx`:

```tsx
onClick: (state) => {
  console.log(`Clicked: ${state}`);
  // Показать статистику по штату
  // Фильтровать грузы
  // и т.д.
}
```
