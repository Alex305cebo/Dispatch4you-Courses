## 🎮 Функции карты:

### Основные:
- **Маркеры траков** - цветные точки показывают где находятся траки
- **Маршруты** - линии показывают куда едет трак
- **Клик по траку** - показывает подробную информацию
- **Легенда** - объясняет цвета статусов (правый верхний угол)

### Интерактивные элементы:
- **Следить за траком** - кнопка внизу справа
- **Zoom** - колёсико мыши или жесты
- **Перемещение** - перетаскивание мышью
- **Тёмная тема** - автоматически включается

### Цвета статусов:
- 🟢 **Зелёный** - везёт груз (loaded)
- 🔵 **Индиго** - едет к погрузке (driving)
- 🟡 **Жёлтый** - на погрузке (at_pickup)
- 🟣 **Фиолетовый** - на разгрузке (at_delivery)
- 🔴 **Красный** - поломка (breakdown)
- 🟠 **Оранжевый** - detention (waiting)
- ⚪ **Серый** - свободен (idle)

## 🔧 Если что-то сломалось:

### Карта не загружается:
1. Откройте консоль (F12)
2. Проверьте ошибки
3. Если видите "RefererNotAllowedMapError":
   - Проверьте настройки API ключа в Google Cloud Console
   - Убедитесь что домены добавлены правильно
   - Подождите 5-10 минут после изменений

### Карта пустая (серый фон):
1. Проверьте что API ключ правильный
2. Проверьте что Maps JavaScript API включен
3. Проверьте консоль на ошибки

### Траки не отображаются:
1. Проверьте что игра запущена (не в меню)
2. Проверьте консоль на ошибки
3. Попробуйте перезагрузить страницу (Ctrl+R)

1. Используйте **HTTP referrers** вместо "None"
