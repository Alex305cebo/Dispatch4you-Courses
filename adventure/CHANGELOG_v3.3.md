## 📦 Технические детали

### Система сохранений

**Поддерживаемые форматы**:
1. Старое сохранение: `dispatcher-game-save`
2. Слот 1: `dispatcher-save-slot-1`
3. Слот 2: `dispatcher-save-slot-2`
4. Слот 3: `dispatcher-save-slot-3`

**Логика загрузки**:
```typescript
// Поиск последнего сохранения
for (let i = 1; i <= 3; i++) {
  const slotData = JSON.parse(localStorage.getItem(`dispatcher-save-slot-${i}`));
  if (slotData.lastSaved > latestTimestamp) {
    latestSave = slotData;
    latestSlot = i;
    latestTimestamp = slotData.lastSaved;
  }
}

// Загрузка найденного
await loadGame(latestSlot);
```

### Проверка версии сохранения

Минимальная поддерживаемая версия: **v6**

Сохранения с версией < 6 автоматически сбрасываются.

```bash

zip -r game-beta-v3.3.0-$(date +%Y%m%d-%H%M%S).zip dist/*
```

### Шаг 4: Загрузка на Hostinger
1. Открыть https://hpanel.hostinger.com
2. File Manager → public_html/adventure/
3. Создать бэкап: переименовать `dist` → `dist.backup.$(date)`
4. Загрузить архив
5. Распаковать в папку `dist/`
6. Очистить кеш: Website → Clear Cache
