# Система автоматического выбора трака

## 🎯 Цель

Упростить процесс назначения груза траку, сделав его быстрым, умным и интуитивным.

## ✨ Основные возможности

### 1. Автоматический выбор
Система автоматически выбирает оптимальный трак на основе:
- 📍 Расстояния до погрузки (deadhead)
- ⏰ Доступных HOS часов
- 😊 Настроения водителя
- 🚛 Статуса трака (idle приоритетнее)

### 2. Интегрированный UI
Всё в одном окне — не нужно переключаться между экранами:
- Переговоры с брокером
- Карточка выбранного трака
- Возможность изменить выбор
- Подтверждение одной кнопкой

### 3. Prebook (резервирование)
Можно резервировать груз для трака который ещё занят:
- Трак на разгрузке → начнёт после разгрузки
- Трак на погрузке → начнёт после погрузки

### 4. Прозрачность
Игрок видит почему выбран именно этот трак:
- Deadhead расстояние
- HOS часы
- Настроение
- Статус

## 🔧 Технические детали

### Алгоритм выбора

```typescript
score = (HOS * 10) + (mood / 10) + statusBonus - (deadhead * 2)
```

- HOS: каждый час = +10 баллов
- Mood: 0-100 → 0-10 баллов
- Status: idle = +50 баллов
- Deadhead: каждая миля = -2 балла

## ❓ FAQ

**Q: Что если нет свободных траков?**  
A: Кнопка "Позвонить брокеру" будет неактивна.

**Q: Можно ли отменить после переговоров?**  
A: Да, нажми "Отмена".

**Q: Как выбрать другой трак?**  
A: Нажми "Изменить" и выбери из списка.

**Q: Что такое Prebook?**  
A: Резервирование груза для занятого трака.
