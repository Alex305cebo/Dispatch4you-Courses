#### 3. **Функциональность**

**Продолжить игру** (если есть сохранение)
- Зелёный градиент (#10b981 → #047857)
- Загружает последнее сохранение
- Переход в игру

**Новая игра**
- Синий градиент (#3b82f6 → #1d4ed8)
- Предупреждение если есть сохранение
- Создаёт новую сессию с 1 траком

**Мои сохранения**
- Открывает SavesManagerPopup
- История последних 5 сохранений
- Загрузка любого сохранения
- Требует авторизацию для облачных сохранений

**Гараж**
- Открывает TruckShopModal
- Покупка новых траков
- Управление флотом
- Бейдж "СКОРО" (в разработке)

**Настройки**
- Открывает SettingsPopup
- Качество графики (Low/Medium/High/Auto)
- Скорость игры (×1, ×2, ×5)
- Сохранение/удаление прогресса

**Профиль**
- Статистика игрока
- Достижения
- Рейтинг
- Бейдж "СКОРО" (в разработке)

**Выход**
- Красная кнопка
- Подтверждение выхода
- Возврат на главную страницу сайта

### 💡 Советы по использованию

#### Для игроков
1. **Первый запуск**
   - Войдите в аккаунт Google
   - Нажмите "Начать игру"
   - Прогресс сохранится автоматически

2. **Продолжение игры**
   - Кнопка "Продолжить" появится автоматически
   - Загружает последнее сохранение
   - Работает на всех устройствах

3. **Облачные сохранения**
   - Войдите в аккаунт
   - Откройте "Мои сохранения"
   - Выберите любое сохранение

4. **Настройки**
   - Выберите качество графики
   - Настройте скорость игры
   - Сохраните прогресс вручную
