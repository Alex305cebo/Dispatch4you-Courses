# 🗺️ Настройки карты Google Maps

## 🛰️ Текущие настройки:

### Вид карты:
- **Тип:** `hybrid` (спутниковый вид с дорогами и названиями)
- **Центр:** США (39.8283°N, 98.5795°W)
- **Начальный zoom:** 5 (вся страна видна)

### Ограничения:
- **Область:** Только континентальная часть США
- **Границы:**
  - Север: 49.384358° (граница с Канадой)
  - Юг: 24.396308° (граница с Мексикой)
  - Запад: -125.0° (Тихий океан)
  - Восток: -66.93457° (Атлантический океан)
- **Минимальный zoom:** 4 (нельзя отдалиться дальше)
- **Максимальный zoom:** 18 (детальный вид улиц)

### Элементы управления:
- ✅ **Zoom control** - кнопки +/- для масштабирования
- ✅ **Map type control** - переключатель типа карты (правый верхний угол)
- ✅ **Scale control** - масштабная линейка
- ✅ **Fullscreen control** - полноэкранный режим
- ❌ **Street View** - отключён (не нужен для игры)
- ❌ **Rotate control** - отключён (не нужен для 2D карты)

## 🎨 Доступные типы карты:

Пользователь может переключаться между:

1. **Roadmap** (Карта) - обычная карта с дорогами
2. **Satellite** (Спутник) - чистый спутниковый снимок
3. **Hybrid** (Гибрид) - спутник + дороги и названия ⭐ **По умолчанию**
4. **Terrain** (Рельеф) - карта с рельефом местности

## 🔧 Как изменить настройки:

### Изменить тип карты по умолчанию:

В файле `GoogleMapView.tsx`, строка ~185:

```typescript
mapTypeId: 'hybrid', // Измените на: 'roadmap', 'satellite', 'terrain'
```

### Изменить границы карты:

В файле `GoogleMapView.tsx`, строки ~178-183:

```typescript
const usaBounds = {
  north: 49.384358,  // Северная граница
  south: 24.396308,  // Южная граница
  west: -125.0,      // Западная граница
  east: -66.93457,   // Восточная граница
};
```

### Убрать ограничение на США:

Удалите или закомментируйте строки ~186-189:

```typescript
// restriction: {
//   latLngBounds: usaBounds,
//   strictBounds: false,
// },
```

### Изменить начальный zoom:

В файле `GoogleMapView.tsx`, строка ~184:

```typescript
zoom: 5, // Измените на: 4 (дальше), 6 (ближе), 7 (ещё ближе)
```

### Отключить переключатель типа карты:

В файле `GoogleMapView.tsx`, строка ~195:

```typescript
mapTypeControl: false, // Было: true
```

## 📊 Сравнение типов карты:

| Тип | Описание | Когда использовать |
|-----|----------|-------------------|
| **roadmap** | Обычная карта | Для навигации, видны все дороги |
| **satellite** | Спутниковый снимок | Для реалистичности, видна местность |
| **hybrid** ⭐ | Спутник + дороги | Лучшее из обоих миров |
| **terrain** | Рельеф местности | Для горных районов |

## 🎯 Почему выбран hybrid:

1. ✅ Реалистичный вид (спутниковые снимки)
2. ✅ Видны дороги и названия городов
3. ✅ Легко ориентироваться
4. ✅ Красиво выглядит
5. ✅ Подходит для игры про грузоперевозки

## 🌍 Почему ограничена только США:

1. ✅ Игра про американские грузоперевозки
2. ✅ Все траки работают только в США
3. ✅ Экономия трафика (не загружаются другие регионы)
4. ✅ Пользователь не может случайно уйти в другую страну
5. ✅ Быстрее загружается

## 💡 Советы:

### Для лучшей производительности:
- Используйте `roadmap` вместо `hybrid` (меньше данных)
- Уменьшите `maxZoom` до 15 (меньше детализация)

### Для лучшего вида:
- Используйте `hybrid` (текущая настройка)
- Увеличьте `maxZoom` до 20 (больше детализация)

### Для большей свободы:
- Уберите `restriction` (можно смотреть весь мир)
- Уберите `minZoom` (можно отдалиться дальше)

## 🔄 Как вернуть обычную карту:

Если спутниковый вид не нравится, измените:

```typescript
mapTypeId: 'roadmap', // Вместо 'hybrid'
mapTypeControl: false, // Скрыть переключатель
```
