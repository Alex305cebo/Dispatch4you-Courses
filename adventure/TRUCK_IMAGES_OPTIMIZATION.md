# Оптимизация картинок траков

## Результаты оптимизации

### До оптимизации
- **Общий размер**: 1323 KB (1.3 MB)
- **Формат**: PNG без сжатия
- **Размер файлов**: 86-162 KB

### После оптимизации
- **Общий размер**: 680 KB (PNG + WebP)
- **Формат**: WebP (основной) + PNG (fallback)
- **Размер PNG**: 34-54 KB
- **Размер WebP**: 14-27 KB
- **Экономия**: 84% для WebP, 60-70% для PNG

## Технические детали

### Использованные инструменты
- **sharp** — библиотека для обработки изображений в Node.js
- **WebP** — современный формат с лучшим сжатием
- **PNG оптимизация** — compressionLevel: 9, effort: 10

### Настройки сжатия
```javascript
// WebP
.webp({ quality: 85, effort: 6 })

// PNG
.png({ quality: 85, compressionLevel: 9, effort: 10 })
```

## Как использовать

### Оптимизация новых картинок
```bash
cd adventure
node scripts/optimize-truck-images.js
```

### Структура файлов
```
Truck Pic/              # Оптимизированные картинки (используются в проекте)
├── 1.png
├── 1.webp
├── 2.png
├── 2.webp
└── ...

Truck Pic Backup/       # Оригинальные картинки (бэкап)
Truck Pic Optimized/    # Временная папка для оптимизации
```

### Использование в коде
```typescript
// Компонент автоматически использует WebP
const getTruckImage = (id: number) => {
  return { uri: `/game/assets/Truck Pic/${id}.webp` };
};
```

## Преимущества

1. **Быстрая загрузка** — картинки загружаются в 3-6 раз быстрее
2. **Экономия трафика** — особенно важно для мобильных устройств
3. **Лучший UX** — игра открывается быстрее
4. **SEO** — Google учитывает скорость загрузки страницы

## Совместимость

- **WebP поддержка**: Chrome, Firefox, Edge, Safari 14+, Opera
- **Fallback на PNG**: для старых браузеров
- **React Native Web**: полная поддержка обоих форматов
