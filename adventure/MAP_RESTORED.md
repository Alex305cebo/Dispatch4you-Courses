## Проблема с кэшем браузера

Если изменения не видны после обновления кода:
1. Ctrl+Shift+R (жёсткая перезагрузка)
2. Открыть DevTools → Application → Clear storage → Clear site data
3. Закрыть все вкладки с игрой и открыть заново
4. В крайнем случае — перезапустить dev сервер

## Почему сломалось

React Native Web не поддерживает прямое использование HTML-элементов (div, canvas) в React Native компонентах. Библиотеки типа pigeon-maps и react-map-gl созданы для обычного React, а не для React Native Web.

Оригинальная карта работает потому что правильно использует Platform.OS === "web" для условного рендера Canvas.
