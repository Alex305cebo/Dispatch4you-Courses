# DUOLINGO-STYLE TUTORIAL SYSTEM
## План микро-обучения для Dispatch Office

### Концепция
Создать систему обучения как в Duolingo:
- **Короткие уроки**: 30-120 секунд каждый
- **Интерактивные вопросы**: Multiple choice с немедленной обратной связью
- **Gamification**: XP, достижения, streaks
- **Прогрессивная сложность**: Beginner → Intermediate → Advanced
- **Практический фокус**: Реальные ситуации из работы диспетчера

## Gamification система

### 🏆 XP (Experience Points)
- Beginner урок: 10-15 XP
- Intermediate урок: 15-20 XP
- Advanced урок: 20-30 XP
- Бонус за streak: +5 XP/день
- Идеальный ответ с первого раза: +5 XP

### 🎖️ Достижения (Achievements)
- **Новичок**: Пройти первые 5 уроков
- **Студент**: Набрать 100 XP
- **Профессионал**: Пройти все уроки модуля
- **Мастер**: Набрать 1000 XP
- **Эксперт**: Пройти все 12 модулей
- **Перфекционист**: Все ответы правильные с первого раза
- **Марафонец**: 7 дней подряд streak
- **Легенда**: 30 дней подряд streak

### 🔥 Streaks (Серии)
- Ежедневное обучение увеличивает streak
- Пропуск дня сбрасывает streak
- Streak freeze: 1 раз в неделю можно пропустить

### 📊 Прогресс
- Процент завершения каждого модуля
- Общий прогресс по всему курсу
- Статистика: сильные и слабые темы
- Время обучения

## UI/UX Дизайн

### Стиль Duolingo
- **Яркие цвета**: Зелёный для правильных ответов, красный для ошибок
- **Анимации**: Конфетти при правильном ответе, тряска при ошибке
- **Прогресс бар**: Визуальный прогресс урока
- **Персонажи**: Аватары ментора, брокера, водителя
- **Звуки**: Приятные звуковые эффекты (опционально)

### Компоненты
1. **LessonCard**: Карточка урока с иконкой, названием, временем, XP
2. **QuestionScreen**: Экран с вопросом и вариантами ответа
3. **FeedbackScreen**: Экран с обратной связью после ответа
4. **ProgressBar**: Прогресс урока (1/5, 2/5, и т.д.)
5. **XPCounter**: Счётчик заработанного XP
6. **StreakDisplay**: Отображение текущего streak
7. **AchievementPopup**: Всплывающее окно при получении достижения

## Метрики успеха

### KPI для системы обучения
- **Completion Rate**: % пользователей, завершивших хотя бы 1 модуль
- **Retention**: % пользователей, возвращающихся на следующий день
- **Average XP**: Средний XP на пользователя
- **Time to Complete**: Среднее время прохождения модуля
- **Accuracy**: % правильных ответов с первого раза

### Целевые показатели
- Completion Rate: >40%
- 7-day Retention: >30%
- Average XP: >200 XP
- Accuracy: >70%
