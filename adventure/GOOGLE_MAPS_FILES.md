## 📄 Описание файлов

### 🗺️ Компоненты карты

#### `GoogleMapView.tsx` — Базовая версия

**Что включено:**
- ✅ Маркеры траков с цветовой кодировкой
- ✅ Прямые линии маршрутов
- ✅ Клик на трак → информация
- ✅ Следование за траком
- ✅ Легенда статусов
- ✅ Тёмная тема

**Когда использовать:**
- Для быстрого старта
- Минимальное потребление API
- Достаточно прямых линий между городами

**API вызовы:**
- 1 загрузка карты при старте
- 0 дополнительных запросов

#### `GoogleMapViewAdvanced.tsx` — Расширенная версия

**Что включено:**
- ✅ Всё из базовой версии +
- ✅ **Directions API** — реальные маршруты по дорогам
- ✅ **Traffic Layer** — пробки в реальном времени
- ✅ Переключение карта/спутник
- ✅ Иконки траков с направлением движения
- ✅ Расчёт расстояния и времени в пути

**Когда использовать:**
- Для production версии
- Нужны реальные маршруты
- Важна точность

**API вызовы:**
- 1 загрузка карты при старте
- N запросов Directions API (по 1 на каждый маршрут)

#### `MapView.tsx` — Старая карта (amCharts)

**Статус:** Резервная копия, не удалена

**Когда использовать:**
- Если нужно откатиться на старую карту
- Для сравнения производительности
- Если Google Maps не подходит

## 🔒 Безопасность

### ⚠️ ВАЖНО: Не коммитьте API ключ!

**Проверьте `.gitignore`:**
```bash
cat .gitignore | grep .env
```

Должно быть:
```
.env
.env.local
.env.*.local
```

**Если случайно закоммитили:**
1. Удалите ключ из истории Git
2. Создайте новый API ключ
3. Удалите старый ключ

## 🎯 Рекомендации

### Для разработки

Используйте **базовую версию** (`GoogleMapView.tsx`):
- Быстрее загружается
- Меньше API вызовов
- Проще отладка

### Для production

Используйте **расширенную версию** (`GoogleMapViewAdvanced.tsx`):
- Реальные маршруты
- Пробки
- Спутниковая карта
- Лучший UX

### Для offline

Используйте **старую карту** (`MapView.tsx`):
- Работает без интернета
- Не требует API ключа
- Легковесная

## 🐛 Проблемы?

### Карта не загружается

1. Проверьте API ключ в `.env`
2. Откройте консоль браузера (F12)
3. Проверьте что API включён в Google Cloud Console

### Маршруты не рисуются

1. Проверьте что Directions API включён
2. Проверьте квоту в Google Cloud Console
3. Откройте консоль и проверьте ошибки

### Нужна помощь?

- Откройте issue в репозитории
- Проверьте документацию
- Спросите в чате проекта
