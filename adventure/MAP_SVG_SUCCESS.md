## Как использовать

### Hover на штат
Просто наведи курсор — штат подсветится автоматически.

### Клик по штату
Можно добавить обработчик в `MapView.tsx`:

```tsx
onClick: (state) => {
  // Показать статистику
  // Фильтровать грузы
  console.log(`Clicked: ${state}`);
}
```

### Изменить цвета
В `MapView.tsx`:

```tsx
fill: hoveredState === state ? '#ТВОЙ_ЦВЕТ' : '#БАЗОВЫЙ_ЦВЕТ',
stroke: hoveredState === state ? '#HOVER_ГРАНИЦА' : '#БАЗОВАЯ_ГРАНИЦА',
```

## Преимущества SVG

1. **Hover из коробки** — не нужна логика проверки координат
2. **CSS стили** — легко кастомизировать
3. **Производительность** — браузер оптимизирует
4. **Доступность** — работает с клавиатурой
5. **Легко расширять** — каждый штат это отдельный элемент
