# ☁️ ОБЛАЧНЫЕ СОХРАНЕНИЯ — FIREBASE

## ✅ Что сделано

Игра теперь **автоматически сохраняет прогресс в облако Firebase**!

### 🎯 Главные возможности

1. **Гибридная система сохранения**
   - Сохраняет локально (localStorage) — быстро, всегда работает
   - Сохраняет в Firebase — синхронизация между устройствами
   - Автоматически выбирает лучший вариант

2. **Синхронизация между устройствами**
   - Начал играть на компьютере → продолжил на телефоне
   - Начал играть на телефоне → продолжил на планшете
   - Все устройства видят одно и то же сохранение

3. **История сохранений**
   - Последние 5 сохранений хранятся в облаке
   - Можно вернуться к любому предыдущему сохранению
   - Удобный UI для просмотра и загрузки

4. **Автоматическое сохранение**
   - Каждые 60 секунд автоматически
   - При критических событиях (назначение груза, доставка)
   - При выходе из игры

## 🎮 Как использовать

### Для игрока

1. **Войти в аккаунт**
   - Нажми "Войти через Google" в навигации
   - Авторизуйся один раз
   - Всё! Теперь прогресс сохраняется автоматически

2. **Просмотр сохранений**
   - Открой ⚙️ Настройки
   - Нажми "💾 Мои сохранения"
   - Увидишь список последних 5 сохранений

3. **Загрузка предыдущего сохранения**
   - В списке сохранений нажми на любое
   - Игра загрузит это сохранение
   - Можно вернуться к любому моменту!

4. **Продолжить на другом устройстве**
   - Войди в тот же аккаунт Google
   - Открой игру
   - Прогресс загрузится автоматически!

### Без авторизации

Если не авторизован:
- Игра сохраняется только локально (localStorage)
- Работает как раньше
- Нет синхронизации между устройствами
- Нет истории сохранений

```
      {saveId1}/
        - (то же самое)
      {saveId2}/
        - (то же самое)
      ...
```

### API функции

```typescript
// Сохранить в Firebase
await saveGameToFirebase(saveData);

// Загрузить из Firebase
const data = await loadGameFromFirebase();

// Получить историю
const saves = await getSaveHistory();

// Загрузить конкретное сохранение
const data = await loadSaveFromHistory(saveId);

// Гибридное сохранение (localStorage + Firebase)
await hybridSave(saveData);

// Гибридная загрузка (Firebase → localStorage)
const data = await hybridLoad();

// Проверить доступность Firebase
const available = isFirebaseAvailable();

// Получить текущего пользователя
const user = await getCurrentUser();
```

### Интеграция в gameStore

```typescript
// Старая версия (только localStorage)
saveGame: () => {
  localStorage.setItem('save', JSON.stringify(data));
}

// Новая версия (гибридная)
saveGame: () => {
  hybridSave(data); // localStorage + Firebase
}

// Загрузка
loadGame: async () => {
  const data = await hybridLoad(); // Firebase → localStorage
  if (data) {
    set(data);
    return true;
  }
  return false;
}
```

## 📊 Преимущества

### До Firebase
- ❌ Сохранение только на одном устройстве
- ❌ Потеря прогресса при очистке браузера
- ❌ Нет истории сохранений
- ❌ Нельзя вернуться к предыдущему моменту

### После Firebase
- ✅ Синхронизация между всеми устройствами
- ✅ Прогресс в облаке — не потеряется
- ✅ История последних 5 сохранений
- ✅ Можно вернуться к любому моменту
- ✅ Автоматическое сохранение каждую минуту

## 🔒 Безопасность

### Firebase Security Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Только авторизованные пользователи
    match /gameSaves/{userId} {
      allow read, write: if request.auth != null 
                         && request.auth.uid == userId;
      
      // История сохранений
      match /history/{saveId} {
        allow read, write: if request.auth != null 
                           && request.auth.uid == userId;
      }
    }
  }
}
```

### Что защищено
- ✅ Каждый пользователь видит только свои сохранения
- ✅ Нельзя читать чужие сохранения
- ✅ Нельзя изменять чужие сохранения
- ✅ Требуется авторизация через Google
