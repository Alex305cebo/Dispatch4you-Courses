### Шаг 1: Открыть Hostinger
```
URL: https://hpanel.hostinger.com
```

### Шаг 2: File Manager
1. Выбрать сайт **dispatch4you.com**
2. Открыть **File Manager**
3. Перейти в `public_html/adventure/`

### Шаг 3: Создать бэкап
```
Переименовать: dist → dist.backup.20260502
```

### Шаг 4: Загрузить архив
1. Нажать **Upload**
2. Выбрать файл: `game-beta-v3.3.0-20260502-151740.zip`
3. Дождаться загрузки (~5-10 минут)

### Шаг 5: Распаковать
1. Правый клик на архив → **Extract**
2. Указать папку: `dist/`
3. Нажать **Extract**

### Шаг 6: Очистить кеш
```
Website → Clear Cache → Clear Cache
```

### Шаг 7: Проверить
```
URL: https://dispatch4you.com/game
Проверить: Версия BETA 3.3, кнопка Continue работает
```
