# ELD График - План исправления

## Проблема
График HOS ELD не отображает линию статусов. Видна только синяя точка.

## Причина
Массив `eldPoints` генерируется неправильно или пустой. Нужно упростить логику генерации данных.

## Решение

### 1. Упростить генерацию данных
Вместо сложной логики с 15-минутными интервалами, использовать простые часовые точки.

### 2. Использовать Canvas API
React Native Web поддерживает Canvas через библиотеку `react-native-canvas`.

### 3. Альтернатива - использовать готовую библиотеку
- `react-native-svg` для рисования графиков
- `victory-native` для готовых графиков

## Временное решение
Показывать упрощенный текстовый вид HOS данных вместо графика, пока не исправим визуализацию.
