# Быстрый старт — Система выбора трака

## Запуск

```bash
cd adventure
npm start
```

Откроется Metro bundler. Нажми:
- `w` — открыть в браузере
- `a` — открыть Android эмулятор
- `i` — открыть iOS симулятор

## Как использовать

### 1. Начни новую смену

1. Открой игру
2. Нажми "Start New Shift"
3. Выбери количество траков (3-5)
4. Дай название сессии

### 2. Найди груз

1. Перейди на вкладку "Load Board"
2. Нажми на любой груз чтобы раскрыть детали
3. Нажми "📞 Позвонить брокеру"

### 3. Проведи переговоры

1. Выбери ставку (например +10%)
2. Брокер ответит
3. Либо прими контроффер, либо предложи свою ставку
4. Дождись успешной сделки

### 4. Выбери трак

После успешных переговоров:

**Автоматический выбор:**
- Система автоматически выбрала лучший трак
- Показана карточка с информацией:
  - Название трака и водитель
  - Текущее местоположение
  - HOS часов
  - Настроение
  - Deadhead расстояние

**Изменить выбор (если нужно):**
1. Нажми "Изменить"
2. Выбери другой трак из списка
3. Список закроется автоматически

**Подтвердить:**
1. Нажми "✅ Подтвердить"
2. Груз назначен!
3. Трак начал движение к погрузке

## Примеры

### Пример 1: Простое назначение

```
1. Груз: Chicago → Dallas, 1092 миль, $2700
2. Переговоры: Успех!
3. Автовыбор: Truck 1047 (Chicago, 0 миль deadhead)
4. Подтверждение: ✅
5. Результат: Трак едет к погрузке
```

### Пример 2: Смена трака

```
1. Груз: Miami → New York, 1280 миль, $3500
2. Переговоры: Успех!
3. Автовыбор: Truck 2023 (Atlanta, 660 миль deadhead)
4. Изменить → Выбрать: Truck 3012 (Savannah, 380 миль)
5. Подтверждение: ✅
6. Результат: Truck 3012 едет к Miami
```

### Пример 3: Prebook

```
1. Груз: Los Angeles → Phoenix, 373 миль, $950
2. Переговоры: Успех!
3. Автовыбор: Truck 4034 (at_delivery в Las Vegas)
4. Статус: "🔄 Заканчивает разгрузку — начнёт после"
5. Подтверждение: ✅
6. Результат: Груз зарезервирован, трак начнёт после разгрузки
```

## Что проверить

✅ Автоматически выбран ближайший трак  
✅ Показан deadhead (если > 0)  
✅ Можно изменить выбор  
✅ Prebook работает для at_delivery/at_pickup  
✅ Кнопка "Подтвердить" назначает груз  
✅ Трак начинает движение к погрузке  

## Частые вопросы

**Q: Что если нет свободных траков?**  
A: Кнопка "📞 Позвонить брокеру" будет неактивна. Дождись пока хотя бы один трак освободится.

**Q: Можно ли отменить после переговоров?**  
A: Да, нажми "Отмена" — груз не будет назначен.

**Q: Как выбрать трак который далеко?**  
A: Нажми "Изменить" и выбери нужный трак из списка. Система покажет deadhead расстояние.

**Q: Что такое Prebook?**  
A: Резервирование груза для трака который ещё занят. Трак автоматически начнёт движение после завершения текущей операции.

**Q: Почему выбран именно этот трак?**  
A: Система выбирает оптимальный трак по формуле:
- Ближайший к погрузке (меньше deadhead)
- Больше HOS часов
- Лучшее настроение водителя
- Idle траки имеют приоритет

## Горячие клавиши

- `Esc` — закрыть окно переговоров
- `Enter` — подтвердить выбор трака (когда кнопка активна)

## Логи

```

✅ Route loaded for T1: 234 points
```

## Поддержка

Если что-то не работает:
1. Проверь консоль на ошибки (F12)
2. Перезагрузи страницу (Ctrl+R)
3. Очисти кэш (Ctrl+Shift+R)
4. Проверь что Metro bundler запущен
