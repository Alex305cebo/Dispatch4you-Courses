# 🚫 Функция отмены груза

## Описание
Реалистичная механика отмены забуканного или активного груза с финансовыми и репутационными последствиями.

## Как использовать

### В игре
1. Откройте панель "Мои грузы" (📦)
2. Для грузов "Ожидают водителя" — нажмите кнопку 🚫 справа
3. Для грузов "В работе" — нажмите "🚫 Отменить груз" внизу карточки
4. Выберите причину отмены
5. Подтвердите решение

## Причины отмены

### 🔧 Поломка трака
- TONU Fee: $150
- Репутация: -5
- Описание: Механическая неисправность

### ⏰ Нарушение HOS
- TONU Fee: $200
- Репутация: -8
- Описание: Водитель не успевает по часам

### 🚨 Экстренная ситуация
- TONU Fee: $100
- Репутация: -3
- Описание: Личная проблема водителя

### 🌨️ Погодные условия
- TONU Fee: $50
- Репутация: -2
- Описание: Опасная погода на маршруте

### 💰 Нашли лучший груз
- TONU Fee: $300
- Репутация: -15
- Описание: Отказ без уважительной причины (самый дорогой!)

## Последствия

1. **TONU Fee** — брокер удерживает деньги за заказ трака
2. **Репутация с брокером** — падает, что влияет на будущие переговоры
3. **Освобождение трака** — водитель становится доступным для новых грузов
4. **Уведомление** — приходит подтверждение отмены

## Технические детали

### Файлы
- `components/CancelLoadModal.tsx` — модальное окно отмены
- `components/MyLoadsPanel.tsx` — кнопки отмены
- `store/gameStore.ts` — функция `cancelLoad()`

### API
```typescript
cancelLoad(loadId: string, params: {
  reason: string;
  tonuFee: number;
  reputationHit: number;
})
```

### Что происходит при отмене
1. Снимается TONU fee с баланса
2. Обновляется репутация с брокером
3. Освобождается трак (если был назначен)
4. Груз удаляется из `activeLoads` и `bookedLoads`
5. Отправляется уведомление

## Реализм
В реальной индустрии грузоперевозок:
- TONU (Truck Ordered Not Used) — стандартная практика
- Размер fee зависит от причины и обстоятельств
- Репутация критически важна для будущих контрактов
- Некоторые причины (погода, поломка) более приемлемы чем другие
