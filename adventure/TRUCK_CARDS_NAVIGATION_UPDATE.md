# Обновление навигации карточек траков

## Изменения

### 1. Перемещение стрелок навигации
- **Было**: Стрелки находились снаружи скролл-контейнера (слева и справа от всей полосы)
- **Стало**: Стрелки перемещены внутрь скролл-контейнера:
  - Стрелка влево (‹) — перед первой карточкой трака
  - Стрелка вправо (›) — после кнопки "Купить трак"

### 2. Автоматическое перемещение выбранной карточки
- При клике на карточку трака она автоматически перемещается на первое место в полосе
- Реализовано через `useMemo` с сортировкой массива траков
- Выбранный трак всегда отображается первым, остальные — в исходном порядке

### 3. Автоскролл к началу
- При выборе трака полоса автоматически скроллится к началу (к выбранной карточке)
- Используется плавная анимация `behavior: 'smooth'`

## Технические детали

### Файл: `TruckCardOverlay.tsx`

#### Добавлен useMemo для сортировки траков:
```typescript
const trucks = useMemo(() => {
  if (!selectedTruckId) return trucksRaw;
  const selectedIdx = trucksRaw.findIndex(t => t.id === selectedTruckId);
  if (selectedIdx < 0) return trucksRaw;
  const selected = trucksRaw[selectedIdx];
  const others = trucksRaw.filter(t => t.id !== selectedTruckId);
  return [selected, ...others];
}, [trucksRaw, selectedTruckId]);
```

#### Обновлён useEffect для автоскролла:
```typescript
useEffect(() => {
  if (!selectedTruckId || !scrollRef.current) return;
  // Выбранная карточка теперь всегда первая, скроллим к началу
  scrollRef.current.scrollTo({ left: 0, behavior: 'smooth' });
}, [selectedTruckId]);
```

#### Стрелки перемещены внутрь скролл-контейнера:
- Стрелка влево добавлена сразу после открытия `<div ref={scrollRef}>`
- Стрелка вправо перемещена перед закрытием скролл-контейнера

## Преимущества

1. **Улучшенная навигация**: Стрелки теперь находятся ближе к карточкам, интуитивно понятнее
2. **Фокус на выбранном траке**: Выбранная карточка всегда на первом месте, легко найти
3. **Плавная анимация**: Автоскролл с плавной анимацией улучшает UX
4. **Консистентность**: Логика работает одинаково на всех устройствах
