## 🔧 Если что-то не работает

### Проблема: Карта не загружается

2. Проверьте ошибки

4. Следуйте инструкциям в ошибке

### Проблема: "RefererNotAllowedMapError"
**Причина:** Домен не разрешён в Google Cloud Console

**Решение:**
1. Откройте [Google Cloud Console](https://console.cloud.google.com/)
2. Перейдите в APIs & Services → Credentials
3. Найдите ваш API ключ
4. Добавьте в HTTP referrers:
   - `localhost:8081/*`
   - `localhost:19006/*`
   - `dispatch4you.com/*`
   - `www.dispatch4you.com/*`

### Проблема: "ApiNotActivatedMapError"
**Причина:** API не включен

**Решение:**
1. Откройте [Google Cloud Console](https://console.cloud.google.com/)
2. Перейдите в APIs & Services → Library
3. Найдите "Maps JavaScript API"
4. Нажмите "Enable"
