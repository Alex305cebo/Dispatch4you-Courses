### 1. Визуальные примеры документов

#### 📄 Rate Confirmation (RateCon)
- Полный пример документа с реальной структурой
- Все ключевые поля: Broker, Carrier, Pickup, Delivery, Cargo, Rate
- Визуальное оформление с цветовой кодировкой
- Highlight важных элементов (ставка, условия оплаты)
- Специальные инструкции и примечания

#### 📦 Bill of Lading (BOL)
- Детальный пример BOL документа
- Shipper, Consignee, Carrier информация
- Таблица с грузом (QTY, Type, Description, Weight)
- Подписи shipper и driver с датами
- Примечания о состоянии груза

  - 12:00-15:00: Переговоры
  - 15:00-17:00: Документация

### 5. Новые CSS стили

#### Document Preview Styles
- `.document-example` - контейнер для примеров
- `.document-preview` - стилизованный документ
- `.doc-header` - шапка документа
- `.doc-section-group` - группа полей
- `.doc-info-row` - строка информации
- `.doc-label` и `.doc-value` - метки и значения
- `.doc-footer` - подвал документа
- `.highlight-row` - выделенные строки (ставка)

#### Process Flow Styles
- `.process-flow` - контейнер для диаграмм
- `.flow-steps` - шаги процесса
- `.flow-step` - отдельный шаг
- `.flow-icon`, `.flow-title`, `.flow-desc` - элементы шага
- Стрелки между шагами (→)
- Hover эффекты

#### Icon Grid Styles
- `.icon-grid` - сетка иконок
- `.icon-item` - элемент с иконкой
- `.icon-item-emoji` - большая иконка
- `.icon-item-label` - подпись
- Hover эффекты

#### Адаптивность
- Мобильная версия для документов
- Вертикальный flow для мобильных
- Адаптивная сетка иконок
