# 🎯 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС

## Шаг 1: Открой Hostinger панель
```
https://hpanel.hostinger.com
```

## Шаг 2: Перейди в Git
Слева в меню → **Git** → **Manage Repositories**

## Шаг 3: Удали старый репозиторий
Найди: `https://github.com/Alex305cebo/dispatch4you-site.git`  
Нажми: **Корзина** (удалить)

## Шаг 4: Добавь новый репозиторий
Нажми: **"Create"** (вверху справа)

Заполни:
```
Repository URL: https://github.com/Alex305cebo/Dispatch4you-Courses.git
Branch: main
Install Path: /
```

Нажми: **"Add repository"**

## Шаг 5: Включи Auto Deployment
Поставь галочку: **"Auto Deployment"**

## Шаг 6: Запусти первый деплой
Нажми: **"Deploy"**

## Шаг 7: Подожди 2-3 минуты
Деплой займет время

## Шаг 8: Проверь сайт
1. Очисти кэш: **Ctrl + Shift + Delete**
2. Открой: **https://dispatch4you.com**
3. Проверь баннер: должен быть "🎓 Профессиональное обучение диспетчеров"

---

## ✅ Готово!

Теперь при каждом `git push origin main` сайт будет автоматически обновляться!

---

**Время:** 5 минут  
**Сложность:** Очень легко
