## 🔧 СПОСОБЫ ЗАГРУЗКИ

### СПОСОБ 1: Через File Manager Hostinger (РЕКОМЕНДУЕТСЯ)

1. **Войдите в Hostinger**
   - Перейдите на https://hpanel.hostinger.com
   - Войдите в свой аккаунт

2. **Откройте File Manager**
   - В панели управления найдите "File Manager"
   - Нажмите "Manage"

3. **Перейдите в public_html**
   - В левой панели найдите папку `public_html`
   - Откройте её

4. **Загрузите shared-nav.css**
   - Найдите существующий файл `shared-nav.css`
   - Нажмите правой кнопкой → "Delete" (удалите старую версию)
   - Нажмите кнопку "Upload" вверху
   - Выберите локальный файл `shared-nav.css`
   - Дождитесь завершения загрузки

5. **Загрузите index.html**
   - Найдите существующий файл `index.html`
   - Нажмите правой кнопкой → "Delete"
   - Нажмите "Upload"
   - Выберите локальный файл `index.html`

6. **Загрузите pages/role.html**
   - Откройте папку `pages`
   - Найдите файл `role.html`
   - Нажмите правой кнопкой → "Delete"
   - Нажмите "Upload"
   - Выберите локальный файл `pages/role.html`

## 🧹 ПОСЛЕ ЗАГРУЗКИ

### 1. Очистите кэш браузера
- **Chrome/Edge**: Ctrl + Shift + Delete → Очистить кэш
- **Firefox**: Ctrl + Shift + Delete → Очистить кэш
- **Safari**: Cmd + Option + E

### 2. Проверьте на разных устройствах
- **Десктоп** (ширина > 1024px): Мобильное меню НЕ должно показываться
- **Планшет** (768px - 1024px): Должно показываться мобильное меню
- **Мобильный** (< 768px): Должно показываться мобильное меню

### 3. Проверьте в разных браузерах
- Chrome
- Firefox
- Safari
- Edge

## ❓ ЕСЛИ ПРОБЛЕМА НЕ РЕШИЛАСЬ

1. **Проверьте версию файла на сервере**
   - Откройте File Manager
   - Откройте `shared-nav.css`
   - Проверьте первую строку (должна быть v3.0)

2. **Проверьте права доступа**
   - Файлы должны иметь права 644
   - Папки должны иметь права 755

3. **Проверьте кэш CDN** (если используется)
   - Cloudflare: Purge Cache
   - Hostinger: Clear Cache в панели управления

4. **Жёсткая перезагрузка страницы**
   - Chrome/Edge: Ctrl + Shift + R
   - Firefox: Ctrl + Shift + R
   - Safari: Cmd + Shift + R
