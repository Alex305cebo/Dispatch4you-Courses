# ⚡ Быстрая настройка Hostinger

## 🎯 Что нужно сделать:

### 1. Удалить старый репозиторий
В Hostinger панели → Git → Удалить `dispatch4you-site.git`

### 2. Добавить новый репозиторий

**Repository URL:**
```
https://github.com/Alex305cebo/Dispatch4you-Courses.git
```

**Branch:**
```
main
```

**Install Path:**
```
/
```

### 3. Включить Auto Deployment
Поставить галочку "Auto Deployment"

### 4. Нажать Deploy
Первый деплой запустить вручную

---

## ✅ Готово!

Теперь при каждом `git push origin main` сайт будет автоматически обновляться!

---

**Время настройки:** 2 минуты  
**Сложность:** Легко
