# Файлы для загрузки на Hostinger

## КРИТИЧЕСКИ ВАЖНЫЕ ФАЙЛЫ (загрузить в первую очередь):

### 1. Навигация (корневая папка):
- `shared-nav.css` - исправлено мобильное меню (display: none !important)
- `nav-loader.js` - версия v2.1

### 2. Главная страница (корневая папка):
- `index.html` - убраны отступы над меню

### 3. Страница role.html (папка pages/):
- `pages/role.html` - градиентные заголовки, кликабельные карточки, компактный timeline

## Как загрузить:

1. Зайти в Hostinger File Manager: https://hpanel.hostinger.com
2. Перейти в папку `public_html`
3. Загрузить файлы из корня проекта в корень `public_html`:
   - shared-nav.css
   - nav-loader.js
   - index.html

4. Перейти в папку `public_html/pages`
5. Загрузить:
   - role.html

## После загрузки:

1. Очистить кэш браузера (Ctrl+Shift+Delete)
2. Обновить страницу с жестким обновлением (Ctrl+F5)
3. Проверить что меню работает правильно

## Проверка версий:

Откройте в браузере DevTools (F12) и проверьте:
- shared-nav.css должен содержать `display: none !important;` для `.mob-menu`
- nav-loader.js должен иметь версию v2.1 в комментарии
