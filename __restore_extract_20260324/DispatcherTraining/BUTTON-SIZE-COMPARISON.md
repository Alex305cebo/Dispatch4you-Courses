## 🎯 СООТВЕТСТВИЕ ДИЗАЙНУ

Теперь кнопки профиля соответствуют кнопкам "Войти" и "Регистрация":

```
Все кнопки navbar:
┌──────────┐  ┌──────────────┐  ┌────────────────────────────────┐  ┌──────────┐
│  Войти   │  │ Регистрация  │  │ 👤 Имя ⚡ XP | Личный кабинет │  │ 🚪 Выйти │
└──────────┘  └──────────────┘  └────────────────────────────────┘  └──────────┘
   6x12px        6x12px                    6x12px                       6x12px
   font 12px     font 12px                 font 12px                    font 12px
```

Единый компактный дизайн на ВСЕХ страницах! 🎉
