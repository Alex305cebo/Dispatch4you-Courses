## 🔧 Что было исправлено

### В файле `shared-nav.css`:

**БЫЛО (неправильно):**
```css
.mob-menu {
    display: flex;  /* ❌ Видно всегда */
}
```

**СТАЛО (правильно):**
```css
.mob-menu {
    display: none;  /* ✅ Скрыто по умолчанию */
    flex-direction: column;
}

@media (max-width: 1024px) {
    .mob-menu {
        display: flex;  /* ✅ Показывается только на мобильных */
    }
}
```
