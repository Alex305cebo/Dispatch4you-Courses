🗺️ Инструкция по отладке карты в game2.html

⚠️ ВАЖНО: Браузер кэширует JavaScript! Нужно принудительно обновить страницу.

Шаг 1: Очистить кэш браузера

Windows/Linux: Нажмите Ctrl + Shift + R или Ctrl + F5

Mac: Нажмите Cmd + Shift + R

Или: Откройте DevTools (F12) → вкладка Network → поставьте галочку "Disable cache" → обновите страницу

Шаг 2: Открыть game2.html

Откройте файл game2.html в браузере

Запустите игру (создайте новую сессию или загрузите сохранение)

Шаг 3: Проверить карту

На карте должны быть видны:

  • ✅ Все штаты США (синие области)
  • ✅ Границы штатов (светло-синие линии)
  • ✅ Дороги (оранжевые линии)
  • ✅ Мичиган полностью (включая Upper Peninsula)
  • ✅ Восточное побережье не обрезано

Шаг 4: Использовать кнопку перерисовки

Справа внизу на карте есть кнопка 🔄

Нажмите её — карта перерисуется и сбросит зум

Шаг 5: Проверить консоль

Откройте консоль браузера (F12 → Console)

Должны быть сообщения:

  • ✅ initMap() завершён, вызываю renderMap()
  • 🗺️ RENDERMAP v2.0 — Штатов: 48
  • 🔄 Принудительная перерисовка через 100ms

Введите в консоли: debugProjection()

Должны появиться координаты углов карты

Что изменилось в коде:

  • ✅ Расширены границы проекции: lng от -126° до -65°, lat от 24° до 50°
  • ✅ Добавлен padding 3% со всех сторон
  • ✅ Штаты стали ярче: rgba(40,60,95,0.95)
  • ✅ Границы штатов ярче и толще: rgba(120,150,190,0.7), 1.8px
  • ✅ Добавлена принудительная очистка canvas
  • ✅ Добавлена кнопка перерисовки 🔄
  • ✅ Добавлена функция отладки debugProjection()
  • ✅ Добавлены console.log для отслеживания

Если карта всё ещё не отображается:

  1. Проверьте что файл us-map-data.js загружается (в консоли не должно быть ошибок 404)
  2. Проверьте что window.US_STATES определён (введите в консоли: Object.keys(window.US_STATES || {}).length)
  3. Проверьте размер canvas (введите в консоли: mapW и mapH)
  4. Нажмите кнопку 🔄 для принудительной перерисовки
