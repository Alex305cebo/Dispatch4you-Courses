# Обновление системы слежения за траками

## Что изменилось

### 1. Автоматическое центрирование карты
- При включении кнопки "Слежение ВКЛ" карта **автоматически центрируется** на выбранном траке
- Центрирование происходит **при каждом обновлении позиции** трака (каждый игровой тик)
- Карта плавно следует за траком используя `panTo()` вместо резкого `setCenter()`

### 2. Меню выбора трака
- Кнопка "🎯 Следить за траком" теперь открывает **выпадающее меню** со списком всех траков
- В меню показывается:
  - Название трака
  - Цветовой индикатор статуса
  - Текущий маршрут (откуда → куда)
- Можно выбрать **конкретный трак** для слежения
- При включенном слежении кнопка показывает название отслеживаемого трака

### 3. Улучшенная логика zoom
- При включении слежения zoom устанавливается на 8 (GoogleMapView) или 10 (GoogleMapViewAdvanced)
- При движении трака zoom **не сбрасывается**, если пользователь приблизил карту
- Zoom обновляется только если он слишком далёкий (< 8 или < 10)

### 4. Оптимизация производительности
- Убран интервал с setInterval (работал каждые 2 секунды)
- Центрирование теперь происходит **реактивно** при обновлении позиции трака
- Меньше нагрузка на браузер, более плавное движение

## Технические детали

### GoogleMapView.tsx
- Добавлен state `followMenuOpen` для управления меню
- Обновлён useEffect для обновления маркеров — теперь включает логику центрирования
- Добавлен отдельный useEffect для начальной центровки при включении слежения
- Обновлён UI кнопки слежения с выпадающим меню

### GoogleMapViewAdvanced.tsx
- Аналогичные изменения для версии с Directions API
- Поддержка стрелок-маркеров с направлением движения
- Zoom установлен на 10 (более детальный вид)

## Как использовать

1. Нажмите на кнопку "🎯 Следить за траком"
2. Выберите трак из выпадающего меню
3. Карта автоматически центрируется на выбранном траке
4. Трак остаётся в центре экрана при движении
5. Для выключения слежения нажмите "🎯 Слежение ВКЛ"
