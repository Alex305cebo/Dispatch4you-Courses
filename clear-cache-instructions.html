🔧 Проблема с кэшем браузера

⚠️ Проблема

Кнопки меню выглядят большими и нестандартными потому что браузер использует СТАРУЮ версию CSS файлов из кэша.

Решение: Нужно очистить кэш браузера!

Способ 1: Жёсткое обновление (САМЫЙ БЫСТРЫЙ)

Нажмите на клавиатуре:

Ctrl + Shift + R

Или:

Ctrl + F5

Это очистит кэш ТОЛЬКО для текущей страницы.

Способ 2: Через DevTools (РЕКОМЕНДУЕТСЯ)

  1. Нажмите F12 чтобы открыть DevTools
  2. Правой кнопкой мыши кликните на кнопку обновления браузера (↻)
  3. Выберите "Empty Cache and Hard Reload" или "Очистить кэш и жёсткое обновление"

Это очистит кэш и перезагрузит страницу.

Способ 3: Полная очистка кэша браузера

Chrome / Edge:

  1. Нажмите Ctrl + Shift + Delete
  2. Выберите "Cached images and files" (Кэшированные изображения и файлы)
  3. Time range: "All time" (Всё время)
  4. Нажмите "Clear data" (Очистить данные)

Firefox:

  1. Нажмите Ctrl + Shift + Delete
  2. Выберите "Cache" (Кэш)
  3. Нажмите "Clear Now" (Очистить сейчас)

Способ 4: Режим инкогнито

Откройте страницу в режиме инкогнито (приватный режим):

Ctrl + Shift + N (Chrome/Edge)

Ctrl + Shift + P (Firefox)

В режиме инкогнито кэш не используется.

✅ Ожидаемый результат

После очистки кэша кнопки должны стать МАЛЕНЬКИМИ:

  • Padding: 6px 12px (вместо 10px 18px)
  • Font-size: 12px (вместо 14px)
  • Все кнопки одинакового размера

🔍 Как проверить что кэш очищен?

  1. Откройте DevTools (F12)
  2. Перейдите на вкладку Network
  3. Обновите страницу (F5)
  4. Найдите файлы:
    • shared-nav.css?v=6.0.1774800000
    • nav-loader.js?v=7.0
