## 🚀 Инструкция по загрузке через Hostinger

### Шаг 1: Подключение к File Manager

1. Зайдите на [hpanel.hostinger.com](https://hpanel.hostinger.com)
2. Выберите ваш сайт **dispatch4you.com**
3. Нажмите **File Manager** в боковом меню
4. Откроется файловый менеджер

### Шаг 2: Навигация к нужной папке

1. В File Manager откройте папку **public_html**
2. Это корневая папка вашего сайта

### Шаг 3: Загрузка файлов

#### Загрузка lesson-audio.css:
1. Найдите файл **lesson-audio.css** в корне public_html
2. Нажмите на него правой кнопкой → **Delete** (удалить старую версию)
3. Нажмите кнопку **Upload Files** вверху
4. Выберите новый файл **lesson-audio.css** с вашего компьютера
5. Дождитесь завершения загрузки

#### Загрузка lesson-audio.js:
1. Найдите файл **lesson-audio.js** в корне public_html
2. Нажмите на него правой кнопкой → **Delete** (удалить старую версию)
3. Нажмите кнопку **Upload Files** вверху
4. Выберите новый файл **lesson-audio.js** с вашего компьютера
5. Дождитесь завершения загрузки

#### Загрузка nav.html:
1. Найдите файл **nav.html** в корне public_html
2. Нажмите на него правой кнопкой → **Delete** (удалить старую версию)
3. Нажмите кнопку **Upload Files** вверху
4. Выберите новый файл **nav.html** с вашего компьютера
5. Дождитесь завершения загрузки

### Шаг 4: Очистка кэша

После загрузки файлов нужно очистить кэш:

1. **Очистка кэша браузера:**
   - Chrome/Edge: `Ctrl + Shift + Delete` → выберите "Изображения и файлы" → Очистить
   - Firefox: `Ctrl + Shift + Delete` → выберите "Кэш" → Очистить
   - Safari: `Cmd + Option + E`

2. **Жесткая перезагрузка страницы:**
   - Windows: `Ctrl + F5` или `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

3. **Очистка кэша Cloudflare (если используется):**
   - Зайдите в панель Cloudflare
   - Перейдите в раздел "Caching"
   - Нажмите "Purge Everything"

Если File Manager не работает, можно использовать FTP-клиент (например, FileZilla):

3. Подключитесь к серверу
