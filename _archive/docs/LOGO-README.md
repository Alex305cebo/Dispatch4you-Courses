# Логотип и Favicon - Инструкция

## Созданные файлы

### 1. **favicon.svg** - Иконка сайта
- Размер: 100×100px
- Используется в браузерах как иконка вкладки
- Современный дизайн с траком, маршрутом и элементами диспетчера

### 2. **logo.svg** - Логотип для навигации
- Размер: 200×60px
- Используется в шапке сайта
- Включает текст "DISPATCH TRAINING USA"

### 3. **apple-touch-icon.svg** - Иконка для iOS
- Размер: 180×180px
- Используется на iPhone/iPad при добавлении на главный экран
- Закругленные углы для iOS стиля

### 4. **og-image.svg** - Изображение для социальных сетей
- Размер: 1200×630px
- Используется в Facebook, LinkedIn, Twitter при шаринге
- Включает полную информацию о курсе

## Конвертация SVG в PNG (опционально)

Если нужны PNG версии для лучшей совместимости:

### Онлайн конвертация:
1. Откройте https://cloudconvert.com/svg-to-png
2. Загрузите SVG файлы
3. Установите качество: 300 DPI
4. Скачайте PNG версии

### Через командную строку (если установлен ImageMagick):
```bash
# Favicon
magick convert -density 300 -background none favicon.svg favicon.png

# Apple Touch Icon
magick convert -density 300 -background none apple-touch-icon.svg apple-touch-icon.png

# OG Image
magick convert -density 300 og-image.svg og-image.png
```

## Где используются файлы

### favicon.svg
- `index.html`: `<link rel="icon" type="image/svg+xml" href="favicon.svg">`
- Все страницы сайта

### logo.svg
- `includes/navigation.html`: В шапке сайта
- Отображается на всех страницах через навигацию

### apple-touch-icon.svg
- `index.html`: `<link rel="apple-touch-icon" href="apple-touch-icon.png">`
- Используется iOS устройствами

### og-image.svg
- `index.html`: `<meta property="og:image" content="https://dispatch4you.com/og-image.png">`
- Социальные сети (Facebook, LinkedIn, Twitter)

## Цветовая схема

### Основные цвета:
- **Cyan/Teal**: `#06b6d4` - основной акцент (трак, текст)
- **Dark Blue**: `#0a1628` - фон
- **Gold**: `#fbbf24` - маршрут, акценты
- **Red**: `#ef4444` - точка назначения
- **Green**: `#10b981` - точка старта
- **Light Gray**: `#e2e8f0` - прицеп трака

### Градиенты:
- Трак: от `#06b6d4` до `#0891b2`
- Маршрут: от `#fbbf24` через `#f59e0b` до `#fbbf24`
- Текст: от `#06b6d4` через `#0ea5e9` до `#06b6d4`

## Дизайн элементы

### Трак (Semi-truck):
- Прицеп с вертикальными полосами
- Кабина с окном
- 3 колеса с деталями
- Фара

### Маршрут:
- Пунктирная линия (дорога)
- Точка старта (зеленая)
- Точка назначения (красная)

### Диспетчер:
- Наушники (headset)
- Символизирует коммуникацию

## Обновление логотипа

Если нужно изменить логотип:

1. Отредактируйте соответствующий SVG файл
2. Если нужны PNG версии - конвертируйте заново
3. Очистите кэш браузера (Ctrl+F5)
4. Проверьте отображение на всех страницах

## Проверка

После обновления проверьте:
- [ ] Favicon отображается в браузере
- [ ] Логотип в навигации корректный
- [ ] OG image работает в социальных сетях (используйте https://www.opengraph.xyz/)
- [ ] Apple touch icon на iOS устройствах
