1. ✅ Явно указать DRY VAN (box trailer)
2. ✅ ОДНОРОДНЫЙ фон БЕЗ уникальных объектов
3. ✅ ВСЕ элементы должны вернуться в исходное положение

WHAT MAKES THIS WORK:
- UNIFORM background (no unique features to "jump")
- CONSTANT speeds (truck, camera, wheels)
- REPETITIVE elements (road markings, landscape)
- IDENTICAL start and end frames
