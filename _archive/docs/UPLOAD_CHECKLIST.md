# ✅ Чеклист загрузки обновлений плеера

## 📋 Перед загрузкой

- [ ] Убедитесь что у вас есть доступ к Hostinger hPanel
- [ ] Откройте папку с проектом на компьютере
- [ ] Найдите 3 файла для загрузки:
  - [ ] `lesson-audio.css`
  - [ ] `lesson-audio.js`
  - [ ] `nav.html`

---

## 🚀 Процесс загрузки

### Шаг 1: Подключение
- [ ] Открыть https://hpanel.hostinger.com
- [ ] Войти в аккаунт
- [ ] Выбрать сайт **dispatch4you.com**
- [ ] Нажать **File Manager**
- [ ] Открыть папку **public_html**

### Шаг 2: Загрузка lesson-audio.css
- [ ] Найти файл `lesson-audio.css` в File Manager
- [ ] Правая кнопка → **Delete** (удалить старую версию)
- [ ] Нажать кнопку **Upload Files** вверху
- [ ] Выбрать новый `lesson-audio.css` с компьютера
- [ ] Дождаться сообщения "Upload successful"

### Шаг 3: Загрузка lesson-audio.js
- [ ] Найти файл `lesson-audio.js` в File Manager
- [ ] Правая кнопка → **Delete** (удалить старую версию)
- [ ] Нажать кнопку **Upload Files** вверху
- [ ] Выбрать новый `lesson-audio.js` с компьютера
- [ ] Дождаться сообщения "Upload successful"

### Шаг 4: Загрузка nav.html
- [ ] Найти файл `nav.html` в File Manager
- [ ] Правая кнопка → **Delete** (удалить старую версию)
- [ ] Нажать кнопку **Upload Files** вверху
- [ ] Выбрать новый `nav.html` с компьютера
- [ ] Дождаться сообщения "Upload successful"

---

## 🧹 Очистка кэша

### В браузере:
- [ ] Нажать `Ctrl + Shift + Delete` (Windows) или `Cmd + Shift + Delete` (Mac)
- [ ] Выбрать "Изображения и файлы в кэше"
- [ ] Нажать "Очистить данные"
- [ ] Закрыть окно

### Жесткая перезагрузка:
- [ ] Открыть сайт dispatch4you.com
- [ ] Нажать `Ctrl + F5` (Windows) или `Cmd + Shift + R` (Mac)
- [ ] Дождаться полной загрузки страницы

### Cloudflare (если используется):
- [ ] Зайти в панель Cloudflare
- [ ] Раздел "Caching"
- [ ] Нажать "Purge Everything"
- [ ] Подтвердить очистку

---

## ✅ Проверка работы

### Тест 1: Логотип
- [ ] Открыть https://dispatch4you.com/pages/documentation.html
- [ ] Проверить что логотип показывает:
  - [ ] "Dispatch4You" (основной текст)
  - [ ] "Курсы Диспетчера" (подзаголовок под ним)

### Тест 2: Плеер появляется
- [ ] Прокрутить страницу вниз
- [ ] Плеер появляется справа
- [ ] Слева от плеера есть кнопка-стрелка ▶

### Тест 3: Сворачивание
- [ ] Нажать на кнопку-стрелку ▶
- [ ] Плеер сворачивается (показывает только кнопку play)
- [ ] Стрелка поворачивается ◀
- [ ] Нажать еще раз
- [ ] Плеер разворачивается обратно

### Тест 4: Индикация воспроизведения
- [ ] Нажать play на любом аудио
- [ ] Кнопка-стрелка меняет цвет на красный
- [ ] Появляется пульсирующая анимация
- [ ] Нажать pause
- [ ] Кнопка возвращается к обычному виду

### Тест 5: Мобильная версия
- [ ] Открыть сайт на телефоне (или режим мобильного в браузере F12)
- [ ] Проверить что плеер работает внизу экрана
- [ ] Проверить что можно свернуть/развернуть

---

## 🐛 Если что-то не работает

### Плеер не сворачивается:
- [ ] Проверить что `lesson-audio.js` загружен
- [ ] Открыть консоль браузера (F12)
- [ ] Проверить нет ли ошибок JavaScript
- [ ] Попробовать режим инкогнито

### Кнопка не меняет цвет:
- [ ] Проверить что `lesson-audio.css` загружен
- [ ] Очистить кэш браузера еще раз
- [ ] Проверить что версия CSS обновилась (посмотреть в исходном коде)

### Логотип неправильный:
- [ ] Проверить что `nav.html` загружен в корень public_html
- [ ] Очистить кэш браузера
- [ ] Проверить что файл не поврежден

### Изменения не видны:
- [ ] Открыть режим инкогнито (Ctrl + Shift + N)
- [ ] Если работает в инкогнито — проблема в кэше
- [ ] Очистить кэш еще раз
- [ ] Перезапустить браузер

Если ничего не помогает:
