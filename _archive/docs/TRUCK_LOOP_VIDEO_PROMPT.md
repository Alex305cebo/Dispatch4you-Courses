# Промпт для генерации видео "Truck Loop" (Gemini Veo 2)

## 🎬 Промпт для Google Gemini (Veo 2)

### Версия 1: Короткая (для быстрой генерации)

```
Create an 8-second perfectly looping video of an American semi-truck 
driving on an interstate highway at sunset. 

CRITICAL: The video MUST loop seamlessly - the last frame at 8 seconds 
must match exactly with the first frame so it can repeat infinitely 
without any visible cut or jump.

Camera follows the truck from behind and slightly to the side. 
Smooth, steady motion. Golden hour lighting. 
Professional trucking documentary style. 
4K quality, 24fps.

The truck should be in the exact same position at the start and end 
of the 8-second clip to ensure perfect looping.
```

### Версия 2: Детальная (для лучшего качества)

```
Create an 8-second PERFECTLY SEAMLESS LOOPING video of a modern American 
semi-truck (18-wheeler) driving on a multi-lane interstate highway during 
golden hour sunset.

⚠️ CRITICAL LOOPING REQUIREMENTS:
- Duration: EXACTLY 8 seconds
- The truck must be in the SAME POSITION at 0:00 and 8:00
- The last frame must PERFECTLY MATCH the first frame
- No visible cut, jump, or discontinuity when looping
- Background elements should also align for seamless loop
- Test: When played on repeat, it should look like one continuous shot

CAMERA:
- Follow the truck from behind-right angle (3/4 rear view)
- Smooth, steady tracking shot at constant speed
- Camera height: slightly elevated, as if from another vehicle
- NO camera shake, NO speed changes, NO zoom
- Camera movement must be perfectly consistent throughout

TRUCK:
- Modern Freightliner or Peterbilt semi-truck
- Dry van trailer (53-foot box trailer)
- Clean, well-maintained appearance
- Truck maintains EXACT same speed throughout (no acceleration/deceleration)

- Same distance from camera throughout entire clip

ENVIRONMENT:
- Wide, empty interstate highway (I-40 or I-80 style)
- 3-4 lanes visible
- Minimal or NO other traffic (to avoid loop discontinuity)
- Clear weather, excellent visibility
- American Southwest or Midwest landscape
- Background should be relatively uniform to help with looping

LIGHTING:
- Golden hour (late afternoon, warm sunlight)
- Sun position should remain consistent (8 seconds is too short for visible sun movement)
- Warm orange and amber tones
- Cinematic color grading
- Consistent lighting throughout - no sudden changes

MOTION:
- Truck maintains perfectly steady 65 mph speed
- Smooth forward motion, no lane changes
- Minimal natural sway from suspension
- Road surface slightly visible (asphalt texture)
- All motion must be cyclical and repeatable

ATMOSPHERE:
- Professional trucking documentary feel
- Cinematic, not amateur
- Slightly desaturated for moody atmosphere
- Film grain: subtle (10-15%)

TECHNICAL:
- Duration: EXACTLY 8 seconds (not 7.9, not 8.1 - exactly 8.0)
- Resolution: 4K (3840×2160)
- Frame rate: 24 fps (cinematic)
- Aspect ratio: 16:9
- SEAMLESS LOOP: First frame = Last frame (CRITICAL!)

LOOPING STRATEGY:
The truck should complete a "visual cycle" in 8 seconds - meaning 
the truck's position, wheel rotation, and all moving elements should 
return to their starting state at the 8-second mark.

STYLE REFERENCES:
- American Truck Simulator cinematics
- Professional trucking documentaries
- Seamless video loops (like Cinemagraphs)
```

## 🔄 Как сделать видео зацикленным (КРИТИЧЕСКИ ВАЖНО!)

### ⚠️ Главное правило для 8-секундного зацикленного видео:

**Кадр на 0:00 секунде = Кадр на 8:00 секунде**

Это означает:
- ✅ Трак в той же позиции
- ✅ Камера в том же положении
- ✅ Колёса в той же фазе вращения
- ✅ Фон в том же месте
- ✅ Освещение идентичное

### Метод 1: В промпте (РЕКОМЕНДУЕТСЯ)

Добавить в промпт явное указание:

```
CRITICAL LOOPING INSTRUCTION:
This is an 8-second video that must loop perfectly.
At exactly 8.00 seconds, the video must return to the exact same state 
as frame 0.00 - same truck position, same camera angle, same background.
When played on repeat (0→8→0→8→0...), it should look like one continuous 
infinite shot with NO visible cuts or jumps.

Think of it as a "boomerang" or "perfect loop" - the end seamlessly 
connects back to the beginning.
```

### Метод 2: Дополнительные инструкции для AI

```
LOOPING STRATEGY FOR 8 SECONDS:
- The truck should travel a distance that creates a visual cycle
- Background elements (road markings, landscape) should repeat
- Camera movement should be perfectly consistent (no acceleration/deceleration)
- Avoid any unique events that can't repeat (like passing a specific landmark)
- Keep the scene uniform and cyclical

WHAT TO AVOID:
❌ Other vehicles passing (they won't be there on loop restart)
❌ Unique landmarks or signs (they break the loop illusion)
❌ Sun position changes (keep lighting constant)
❌ Camera speed changes (must be perfectly steady)
❌ Truck lane changes (position must stay consistent)
```

### Метод 2: Постобработка (если AI не зациклил идеально)

**Если видео НЕ зациклено идеально, можно исправить:**

#### Вариант A: Кроссфейд между концом и началом

**В Adobe Premiere / DaVinci Resolve:**
1. Импортировать видео
2. Дублировать клип на таймлайне (положить 2 копии подряд)
3. Добавить эффект "Cross Dissolve" между концом первого и началом второго
4. Длительность перехода: 0.3-0.5 секунды
5. Вырезать второй клип после перехода
6. Экспортировать

**В FFmpeg (командная строка):**
```bash
# Создать плавный переход между концом и началом
ffmpeg -i input.mp4 -filter_complex \
"[0:v]split[main][dup]; \
[dup]trim=start=7:end=8,setpts=PTS-STARTPTS[end]; \
[main]trim=start=0:end=1,setpts=PTS-STARTPTS[start]; \
[end][start]xfade=transition=fade:duration=0.5:offset=0.5[fade]; \
[main][fade]concat=n=2:v=1[out]" \
-map "[out]" -t 8 output_loop.mp4
```

#### Вариант B: Обрезать до идеального зацикливания

Если видео почти зациклено, но не идеально:

```bash
# Найти лучшую точку зацикливания (например, на 7.8 сек вместо 8.0)
ffmpeg -i input.mp4 -t 7.8 -c copy output_loop.mp4
```

#### Вариант C: Реверс + зацикливание (если совсем не получилось)

```bash
# Создать эффект "туда-обратно" (boomerang)
ffmpeg -i input.mp4 -filter_complex \
"[0:v]split[a][b]; \
[a]trim=0:8,setpts=PTS-STARTPTS[forward]; \
[b]trim=0:8,setpts=PTS-STARTPTS,reverse[backward]; \
[forward][backward]concat=n=2:v=1[out]" \
-map "[out]" output_boomerang.mp4
```

## ✅ Как проверить что видео зациклено правильно

### Тест 1: Визуальная проверка
1. Открыть видео в плеере с функцией loop (VLC, QuickTime)
2. Включить повтор (loop)
3. Смотреть на стык между концом и началом
4. ✅ Не должно быть видимого "прыжка" или "скачка"

### Тест 2: Покадровая проверка
1. Открыть видео в видеоредакторе
2. Перейти к последнему кадру (7.99 сек)
3. Сравнить с первым кадром (0.00 сек)
4. ✅ Трак должен быть в той же позиции
5. ✅ Фон должен совпадать

### Тест 3: В браузере
```html
<video autoplay muted loop>
  <source src="Truck_loop.mp4" type="video/mp4">
</video>
```
Открыть в браузере и смотреть несколько циклов.

### Тест 4: Наложение кадров
В Photoshop или After Effects:
1. Импортировать первый кадр (0.00)
2. Импортировать последний кадр (8.00)
3. Наложить друг на друга с прозрачностью 50%
4. ✅ Если зациклено идеально — изображения совпадут

## 🎯 Признаки ИДЕАЛЬНО зацикленного видео

✅ **Хорошее зацикливание:**
- При повторе невозможно определить где конец и начало
- Трак движется плавно без "телепортации"
- Фон течёт непрерывно
- Колёса вращаются без скачков
- Можно смотреть бесконечно без дискомфорта

❌ **Плохое зацикливание:**
- Видимый "прыжок" трака при повторе
- Фон "скачет" или "дёргается"
- Колёса "телепортируются" в другую фазу
- Освещение меняется резко
- Глаз "спотыкается" на стыке

## 💡 Советы для идеального зацикливания

### Совет 1: Минимализм
Чем проще сцена, тем легче зациклить:
- ✅ Пустая дорога (без других машин)
- ✅ Однородный фон (пустыня, равнина)
- ✅ Постоянная скорость
- ❌ Сложный трафик
- ❌ Уникальные ориентиры
- ❌ Меняющееся освещение

### Совет 2: Короткая длительность
8 секунд — оптимально:
- ✅ Достаточно для плавного движения
- ✅ Легко зациклить
- ✅ Маленький размер файла
- ❌ 30 секунд — слишком долго, сложно зациклить

### Совет 3: Постоянная скорость
- ✅ Трак едет с одной скоростью
- ✅ Камера движется равномерно
- ❌ Ускорения/замедления ломают цикл

### Совет 4: Тестирование
Всегда проверяйте зацикливание перед использованием:
```bash
# Создать тестовое видео с 10 повторами
ffmpeg -stream_loop 10 -i Truck_loop.mp4 -c copy test_10_loops.mp4
```
Смотрите на стыки — они должны быть невидимы.

## 📦 Оптимизация для веб

После генерации видео, оптимизируйте его для веба:

### Сжатие (FFmpeg):
```bash
# Сжать до 1-2 MB для быстрой загрузки
ffmpeg -i Truck_loop.mp4 \
  -vcodec h264 \
  -crf 28 \
  -preset slow \
  -vf scale=1920:1080 \
  -movflags +faststart \
  Truck_loop_compressed.mp4
```

### Конвертация в WebM (лучше для веба):
```bash
ffmpeg -i Truck_loop.mp4 \
  -c:v libvpx-vp9 \
  -b:v 1M \
  -vf scale=1920:1080 \
  Truck_loop.webm
```

### Создание превью (poster):
```bash
# Извлечь первый кадр как изображение
ffmpeg -i Truck_loop.mp4 \
  -vframes 1 \
  -vf scale=1920:1080 \
  truck_poster.jpg
```
