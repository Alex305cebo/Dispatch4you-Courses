# Быстрый режим карты (опционально)

Если карта всё ещё грузится медленно, можно включить **быстрый режим**.

## Как включить быстрый режим

### Вариант 1: Изменить тип карты по умолчанию

В файле `GoogleMapView.tsx` найдите строку:
```typescript
mapTypeId: 'hybrid', // Спутниковый вид с дорогами
```

Замените на:
```typescript
mapTypeId: 'roadmap', // Обычная карта (быстрее)
```

### Вариант 2: Изменить в переменной состояния

Найдите строку:
```typescript
const [currentMapType, setCurrentMapType] = useState<'roadmap' | 'satellite' | 'hybrid' | 'terrain'>('hybrid');
```

Замените на:
```typescript
const [currentMapType, setCurrentMapType] = useState<'roadmap' | 'satellite' | 'hybrid' | 'terrain'>('roadmap');
```

## Сравнение типов карт

### 🌐 Hybrid (по умолчанию)
- **Скорость:** Средняя
- **Размер данных:** ~2-3 MB
- **Плюсы:** Красиво, реалистично, видны дороги и местность
- **Минусы:** Медленнее загружается

### 🗺️ Roadmap (быстрый режим)
- **Скорость:** Быстрая
- **Размер данных:** ~500 KB
- **Плюсы:** Быстрая загрузка, чёткие дороги
- **Минусы:** Менее реалистично

### 🛰️ Satellite
- **Скорость:** Медленная
- **Размер данных:** ~3-4 MB
- **Плюсы:** Максимально реалистично
- **Минусы:** Самая медленная загрузка

### ⛰️ Terrain
- **Скорость:** Средняя
- **Размер данных:** ~1-2 MB
- **Плюсы:** Показывает рельеф местности
- **Минусы:** Не всегда нужен для диспетчера

## Рекомендации

### Для медленного интернета
```typescript
mapTypeId: 'roadmap'
zoom: 4
```

### Для среднего интернета (по умолчанию)
```typescript
mapTypeId: 'hybrid'
zoom: 4.5
```

### Для быстрого интернета
```typescript
mapTypeId: 'satellite'
zoom: 5
```

## Переключение на лету

Пользователь может переключать тип карты через кнопку в левом нижнем углу:
- 🌐 Гибрид
- 🗺️ Карта
- 🛰️ Спутник
- ⛰️ Рельеф

Выбранный тип сохраняется в состоянии компонента.
