# 🎓 Tutorial System v1.0.0

**Модульная система нарративного обучения для DispatcherTraining**

---

## 🚀 Что это?

Полнофункциональная система обучения с персонажем-наставником, которая учит игрока через контекстные задачи, а не через скучные туториалы.

**Ключевые особенности:**
- ✅ Модульная архитектура (State Machine + UI + Data)
- ✅ 11 готовых шагов обучения
- ✅ Интеграция с существующим UI
- ✅ Автоматическое сохранение прогресса
- ✅ Адаптивный дизайн (desktop/mobile)
- ✅ Полная документация (~2,400 строк)

---

## 📦 Что включено

### Код (~1,650 строк)
- `TutorialStateMachine.js` — State Machine (FSM)
- `MentorshipData.js` — Данные обучения (11 шагов)
- `TutorialUIController.js` — UI контроллер
- `TutorialManager.js` — Главный оркестратор
- `integration.js` — Интеграция с игрой

### Документация (~2,400 строк)
- `README.md` — Полная документация API
- `ARCHITECTURE.md` — Архитектурная документация
- `EXAMPLES.md` — 19 примеров кода
- `INTEGRATION_GUIDE.md` — Руководство по интеграции
- `VISUAL_GUIDE.md` — Визуальное руководство
- `CHANGELOG.md` — История изменений

### Демо
- `tutorial-demo.html` — Рабочая демонстрация

---

## ⚡ Быстрый старт

### 1. Добавь в игру

```html
<script type="module">
  import { initializeAll } from './js/tutorial/integration.js';
  initializeAll();
</script>
```

### 2. Добавь ID к кнопкам

```html
<button id="start-shift-btn">Начать смену</button>
<button id="accept-load-btn">Принять груз</button>
```

### 3. Готово!

Система автоматически запустит обучение для новых пользователей.

## 🎯 11 шагов обучения

1. Приветствие в офисе
2. Проверка флота траков
3. Проверка входящей почты
4. Проверка Rate Confirmation
5. Принятие груза
6. Отправка задания водителю
7. Мониторинг прогресса
8. Вопрос от водителя
9. Решение проблемы
10. Доставка завершена
11. Обучение завершено

## 🏗️ Архитектура

```
TutorialManager (Оркестратор)
    ├── TutorialStateMachine (FSM логика)
    ├── TutorialUIController (Визуализация)
    └── MentorshipData (Данные)
```

**Паттерны:** State Machine, Observer, Event Bus, Facade, Strategy, Data-Driven
