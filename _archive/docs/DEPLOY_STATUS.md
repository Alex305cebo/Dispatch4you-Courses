## 🐛 Если что-то не работает

### Изменения не видны:
1. Подождите 2-3 минуты после завершения деплоя
2. Очистите кэш браузера: `Ctrl + Shift + Delete`
3. Жесткая перезагрузка: `Ctrl + F5`
4. Попробуйте режим инкогнито: `Ctrl + Shift + N`

### GitHub Actions завис:
1. Проверьте статус на странице Actions
2. Если есть ошибки — посмотрите логи
3. При необходимости перезапустите workflow

### Плеер не работает:
1. Откройте консоль браузера (F12)
2. Проверьте нет ли ошибок JavaScript
3. Убедитесь что файлы загрузились на сервер
4. Проверьте что версия файлов обновилась
