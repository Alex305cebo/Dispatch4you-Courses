## 🎯 Ключевые компоненты

### 1. GameState (gameStore.ts)
```typescript
interface GameState {
  currentSlotId?: number;  // ← Новое поле
  phase: 'menu' | 'playing' | 'day_end' | 'shift_end';
  day: number;
  gameMinute: number;
  sessionName: string;
  balance: number;

  // ...
```

```typescript
interface SaveSlot {
```

## 🔐 Безопасность и надёжность

### Fallback механизм
```
loadGame(slotId)
    ↓
Пробуем загрузить из слота
    ↓
Слот пуст?
    ↓ Да
Пробуем гибридную загрузку (Firebase + localStorage)
    ↓
Найдено?
    ↓ Да
Загружаем
    ↓ Нет
Возвращаем false
```

### Версионирование
```typescript
if (!saveData.version || saveData.version < 6) {
  // Сброс старых сохранений
  localStorage.removeItem('dispatcher-game-save');
  clearAllSaves();
  return false;
}
```

### Валидация данных
```typescript
try {
  const data = JSON.parse(raw);
  if (data.version >= 6 && data.phase === 'playing') {
    // Валидное сохранение
  }
} catch {
  // Повреждённое сохранение
  alert('Ошибка загрузки сохранения');
}
```
