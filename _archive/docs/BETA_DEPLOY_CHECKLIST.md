## 🚀 Деплой на Hostinger

### Шаг 1: Вход в панель
- [ ] Открыть https://hpanel.hostinger.com
- [ ] Выбрать сайт dispatch4you.com
- [ ] Открыть File Manager

### Шаг 2: Бэкап текущей версии
- [ ] Перейти в `public_html/adventure/`
- [ ] Переименовать папку `dist` → `dist.backup.20260418`
- [ ] Убедиться что бэкап создан

### Шаг 3: Загрузка новой версии
- [ ] Загрузить файл `game-beta-v2.2.0-20260418-190441.zip`
- [ ] Загрузить в папку `public_html/adventure/`
- [ ] Дождаться завершения загрузки (5.18 MB)

### Шаг 4: Распаковка
- [ ] Кликнуть правой кнопкой на архив
- [ ] Выбрать "Extract"
- [ ] Указать папку назначения: `dist/`
- [ ] Дождаться распаковки

### Шаг 5: Проверка файлов
- [ ] Открыть папку `public_html/adventure/dist/`
- [ ] Проверить наличие:
  - [ ] index.html
  - [ ] interactive-feedback.css
  - [ ] interactive-feedback.js
  - [ ] _expo/ (папка)
  - [ ] assets/ (папка)

### Шаг 6: Очистка кэша
- [ ] Вернуться в Hostinger Panel
- [ ] Website → Clear Cache
- [ ] Дождаться подтверждения
