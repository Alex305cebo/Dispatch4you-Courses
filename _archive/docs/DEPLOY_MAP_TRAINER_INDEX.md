# 🚀 Инструкция по деплою index.html для USA Map Trainer

## 📁 Файл для загрузки
**Локальный путь:** `c:\DispatcherTraining\map-trainer\index.html`

## 🎯 Куда загружать
**Путь на хостинге:** `/public_html/map-trainer/index.html`

## 📋 Пошаговая инструкция

### Способ 1: Через File Manager Hostinger (Рекомендуется)

1. **Войдите в Hostinger**
   - Откройте https://hpanel.hostinger.com/
   - Войдите в аккаунт

2. **Откройте File Manager**
   - В панели управления найдите "File Manager" (Файловый менеджер)
   - Нажмите "Manage" или "Open"

3. **Перейдите в нужную папку**
   - Откройте папку `public_html`
   - Откройте папку `map-trainer`

4. **Замените файл**
   - Найдите файл `index.html`
   - Нажмите правой кнопкой → Delete (или переименуйте в `index.html.old`)
   - Нажмите кнопку "Upload" (Загрузить)
   - Выберите файл: `c:\DispatcherTraining\map-trainer\index.html`
   - Дождитесь завершения загрузки

5. **Проверьте права доступа**
   - Убедитесь что права на файл: `644` (rw-r--r--)

### Способ 2: Через FTP

1. **Подключитесь к FTP**
   - Host: `ftp.dispatch4you.com` (или IP сервера)
   - Username: ваш FTP логин
   - Password: ваш FTP пароль
   - Port: 21

2. **Перейдите в папку**
   - Откройте `/public_html/map-trainer/`

3. **Загрузите файл**
   - Перетащите `index.html` из локальной папки
   - Подтвердите замену файла

## 📞 Если что-то пошло не так

1. **Файл не загружается**
   - Проверьте права доступа к папке
   - Попробуйте через другой браузер

2. **Карточка не появляется**
   - Проверьте исходный код страницы (Ctrl+U)
   - Убедитесь что meta-теги присутствуют
   - Попробуйте в новом чате Telegram

3. **Приложение не работает**
   - Убедитесь что папка `assets/` на месте
   - Проверьте что файлы JS и CSS доступны
