### ✅ 2. Автооткрытие секции "Курс обучения" в мобильном меню
**Статус:** РАБОТАЕТ ✅

**Файл:** `nav-loader.js` v7.1
```javascript
function openMenu() {
    var menu = document.getElementById('mobMenu');
    var overlay = document.getElementById('mobOverlay');
    if (menu) menu.classList.add('active');
    if (overlay) overlay.classList.add('active');
    document.body.style.overflow = 'hidden';
    
    // Автоматически раскрываем первую секцию "Курс обучения"
    setTimeout(function() {
        var firstAccordion = document.querySelector('.mob-acc');
        if (firstAccordion && !firstAccordion.classList.contains('open')) {
            firstAccordion.classList.add('open');
        }
    }, 100);
}
```

**Результат:** При открытии бургер-меню секция "📚 Курс обучения" автоматически раскрыта

### ✅ 5. Контент не заходит под navbar
**Статус:** ИСПРАВЛЕНО ✅

**Файл:** `index.html`

**Проблема:** Заголовок "Станьте Диспетчером..." заходил под navbar

**Решение:** Добавлен padding-top для hero секции
```css
/* Основной стиль */
.hero {
  padding: 84px 0 20px 0; /* 64px navbar + 20px spacing */
}

/* Мобильные устройства */
@media (max-width: 768px) {
  .hero {
    padding: 84px 0 60px;
  }
}

@media (max-width: 480px) {
  .hero {
    padding: 84px 0 50px;
  }
}

@media (max-width: 360px) {
  .hero {
    padding: 84px 0 40px;
  }
}

/* Compact spacing override */
.hero { 
  padding-top: 84px !important; 
  padding-bottom: 20px !important; 
}
```

**Результат:** Контент начинается после navbar с правильным отступом
