# 🚨 РУЧНАЯ ЗАГРУЗКА ФАЙЛОВ - Пошаговая инструкция

## Проблема
Автодеплой с GitHub на Hostinger не работает или не настроен.  
Файлы отправлены в GitHub, но не появились на сайте.

## Решение
Загрузить файлы вручную через Hostinger File Manager.

---

## 📋 ШАГ 1: Откройте Hostinger (2 минуты)

1. Откройте браузер
2. Перейдите на: **https://hpanel.hostinger.com/**
3. Войдите в аккаунт (email + пароль)
4. Выберите ваш хостинг-план

---

## 📋 ШАГ 2: Откройте File Manager (1 минута)

1. В левом меню найдите **Files**
2. Нажмите **File Manager**
3. Откроется файловый менеджер в новой вкладке
4. В левой панели найдите папку **public_html**
5. Кликните на неё (откроется содержимое)

---

## 📋 ШАГ 3: Загрузите robots.txt (2 минуты)

### 3.1 Найдите файл на компьютере
- Откройте папку: `C:\DispatcherTraining\`
- Найдите файл: `robots.txt`

### 3.2 Загрузите на сервер
1. В File Manager нажмите кнопку **Upload Files** (вверху справа)
2. Выберите файл `robots.txt` с компьютера
3. Нажмите **Open** (Открыть)
4. Дождитесь завершения загрузки (зелёная галочка)

### 3.3 Проверьте
- В списке файлов должен появиться `robots.txt`
- Размер: ~1 KB

---

## 🆘 Если возникли проблемы

### Проблема: Не могу найти File Manager
**Решение:**
1. В hPanel слева найдите раздел **Files**
2. Нажмите на него
3. Выберите **File Manager**
4. Если не видите - свяжитесь с поддержкой Hostinger

### Проблема: Не могу загрузить файл
**Решение:**
1. Проверьте размер файла (должен быть < 10 MB)
2. Проверьте что вы в папке public_html
3. Попробуйте другой браузер
4. Очистите кеш браузера

### Проблема: Файл загрузился, но не работает (404)
**Решение:**
1. Проверьте что файл в корне public_html (не в подпапке)
2. Проверьте имя файла (должно быть точно: robots.txt, не Robots.txt)
3. Очистите кеш браузера (Ctrl+Shift+Delete)
4. Подождите 5-10 минут

### Проблема: Не вижу .htaccess
**Решение:**
1. В File Manager нажмите Settings (шестерёнка)
2. Включите **Show Hidden Files**
3. Нажмите Save
4. Обновите страницу
