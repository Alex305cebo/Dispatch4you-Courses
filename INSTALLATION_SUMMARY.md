## 🎯 How It Works

1. **User clicks** red microphone icon 🔴 in Chrome toolbar
2. **User holds** button and **speaks in Russian**
3. **User releases** button
4. **Chrome recognizes** speech using Web Speech API
5. **Extension translates** to English via Google Translate
6. **Result auto-copied** to clipboard
7. **User pastes** (Ctrl+V) anywhere!

**Keyboard shortcut:** `Ctrl+Shift+V`

## 🛠️ Troubleshooting Guide (for user)

### "Extension doesn't load"
- Make sure Developer mode is ON
- Try restarting Chrome
- Re-extract ZIP (files might be corrupted)

### "No microphone access"
- Click lock icon in Chrome address bar
- Allow microphone permission
- Check Windows microphone settings

### "Speech not recognized"
- Speak louder and clearer
- Test microphone in Windows settings
- Make sure correct microphone is selected

### "Translation fails"
- Check internet connection
- Try again (service might be busy)
- Wait a few seconds between attempts
