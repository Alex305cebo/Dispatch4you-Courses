# 🔐 Система контроля доступа к курсам

## 📊 Уровни подписки

### 1. FREE (Бесплатный)
**Доступ:**
- ✅ Главная страница
- ✅ О нас, FAQ, Контакты
- ✅ Модуль 1 (полностью) - Введение в диспетчинг
- ✅ Первый сектор каждого модуля (preview)
- ✅ Базовые инструменты (ограниченно)

**Ограничения:**
- ❌ Модули 2-10 (только preview)
- ❌ Тесты и сертификаты
- ❌ Симулятор
- ❌ Dispatcher Cards
- ❌ Load Finder (ограничен 5 поисками/день)

---

### 2. TRIAL (Пробный - 7 дней)
**Доступ:**
- ✅ Все из FREE
- ✅ Модули 1-3 (полностью)
- ✅ Базовые тесты
- ✅ Симулятор (ограничен 10 сессиями)
- ✅ Load Finder (20 поисков/день)

**Ограничения:**
- ❌ Модули 4-10
- ❌ Сертификаты
- ❌ Dispatcher Cards
- ⏰ Истекает через 7 дней

---

### 3. BASIC ($49/месяц)
**Доступ:**
- ✅ Все из TRIAL
- ✅ Модули 1-5 (полностью)
- ✅ Все тесты модулей 1-5
- ✅ Симулятор (50 сессий/месяц)
- ✅ Load Finder (100 поисков/день)
- ✅ Email поддержка

**Ограничения:**
- ❌ Модули 6-10
- ❌ Сертификаты
- ❌ Dispatcher Cards
- ❌ Приоритетная поддержка

---

### 4. PRO ($99/месяц) ⭐ ПОПУЛЯРНЫЙ
**Доступ:**
- ✅ Все модули 1-10 (полностью)
- ✅ Все тесты и экзамены
- ✅ Сертификаты по модулям
- ✅ Симулятор (безлимит)
- ✅ Load Finder (безлимит)
- ✅ Dispatcher Cards (базовый набор)
- ✅ Приоритетная email поддержка
- ✅ Доступ к закрытому Telegram каналу

**Ограничения:**
- ❌ Персональный наставник
- ❌ Расширенные Dispatcher Cards
- ❌ Помощь с трудоустройством

---

### 5. PREMIUM ($199/месяц) 💎
**Доступ:**
- ✅ ВСЁ из PRO
- ✅ Персональный наставник (2 часа/месяц)
- ✅ Dispatcher Cards (полный набор + обновления)
- ✅ Помощь с составлением резюме
- ✅ Помощь с трудоустройством
- ✅ Доступ к базе вакансий
- ✅ Приоритетная поддержка 24/7
- ✅ Индивидуальные консультации
- ✅ Lifetime доступ к материалам (после 6 месяцев)

## 🔧 Техническая реализация

### 1. Структура данных пользователя (Firestore)

```javascript
{
  uid: "user123",
  email: "user@example.com",
  firstName: "Иван",
  lastName: "Петров",
  
  // Подписка
  subscription: {
    type: "pro", // free, trial, basic, pro, premium
    status: "active", // active, expired, cancelled
    startDate: "2026-03-01",
    endDate: "2026-04-01",
    autoRenew: true,
    paymentMethod: "stripe"
  },
  
  // Trial
  trial: {
    used: true,
    startDate: "2026-02-20",
    endDate: "2026-02-27"
  },
  
  // Прогресс
  progress: {
    completedModules: [1, 2, 3],
    completedSectors: [
      { module: 1, sector: 1, completedAt: "2026-02-21" },
      { module: 1, sector: 2, completedAt: "2026-02-22" }
    ],
    testScores: [
      { module: 1, score: 85, passedAt: "2026-02-23" }
    ],
    certificates: [
      { module: 1, issuedAt: "2026-02-23", certificateId: "CERT-001" }
    ]
  },
  
  // Использование
  usage: {
    simulatorSessions: 15,
    loadFinderSearches: 45,
    lastActive: "2026-03-23"
  }
}
```

### 2. Файлы для создания

#### `access-control.js` - Основная логика контроля доступа
```javascript
// Проверка доступа к модулю
function hasModuleAccess(userSubscription, moduleNumber)

// Проверка доступа к сектору
function hasSectorAccess(userSubscription, moduleNumber, sectorNumber)

// Проверка доступа к инструменту
function hasToolAccess(userSubscription, toolName)

// Проверка лимитов использования
function checkUsageLimit(userId, feature)
```

#### `subscription-manager.js` - Управление подписками
```javascript
// Создание подписки
function createSubscription(userId, planType)

// Обновление подписки
function upgradeSubscription(userId, newPlanType)

// Отмена подписки
function cancelSubscription(userId)

// Проверка статуса подписки
function checkSubscriptionStatus(userId)
```

#### `paywall.js` - UI для блокировки контента
```javascript
// Показать paywall
function showPaywall(requiredPlan, currentPlan)

// Показать upgrade предложение
function showUpgradeOffer(feature)
```
