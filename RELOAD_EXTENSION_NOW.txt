⚠️  ВАЖНО: После изменений нужно ПЕРЕЗАГРУЗИТЬ расширение!


═══════════════════════════════════════════════════════════════════
ШАГ 1: Откройте chrome://extensions/
═══════════════════════════════════════════════════════════════════

1. Скопируйте:  chrome://extensions/
2. Вставьте в адресную строку Chrome
3. Нажмите Enter


═══════════════════════════════════════════════════════════════════
ШАГ 2: Найдите Voice-to-Chat
═══════════════════════════════════════════════════════════════════

Найдите карточку "Voice-to-Chat: RU → EN"


═══════════════════════════════════════════════════════════════════
ШАГ 3: Перезагрузите расширение
═══════════════════════════════════════════════════════════════════

Нажмите кнопку обновления ⟳ (круговая стрелка)

Или:
1. Выключите тумблер (OFF)
2. Включите обратно (ON)


═══════════════════════════════════════════════════════════════════
ШАГ 4: Откройте расширение заново
═══════════════════════════════════════════════════════════════════

Кликните на иконку 🔴 Voice-to-Chat


═══════════════════════════════════════════════════════════════════
ЧТО УВИДИТЕ:
═══════════════════════════════════════════════════════════════════

ВАРИАНТ А: "✅ Готов к записи (микрофон разрешён)"
   → ВСЁ РАБОТАЕТ! Жмите кнопку и говорите

ВАРИАНТ Б: "⚠️ Нажмите кнопку - Chrome запросит доступ"
   → Нажмите кнопку записи
   → Chrome покажет запрос
   → Нажмите "Разрешить"

ВАРИАНТ В: "НУЖНО РАЗРЕШЕНИЕ!"
   → Увидите БОЛЬШУЮ ЗЕЛЁНУЮ КНОПКУ:
     "🔑 РАЗРЕШИТЬ ДОСТУП К МИКРОФОНУ"
   → Нажмите её
   → Chrome покажет запрос
   → Нажмите "Разрешить"


═══════════════════════════════════════════════════════════════════
ЕСЛИ ВСПЛЫВАЮЩИЙ ЗАПРОС НЕ ПОЯВЛЯЕТСЯ:
═══════════════════════════════════════════════════════════════════

1. Посмотрите в АДРЕСНУЮ СТРОКУ (слева)
2. Там должна быть иконка микрофона 🎤
3. Кликните на неё
4. Выберите "Разрешить"
5. Перезагрузите расширение снова


═══════════════════════════════════════════════════════════════════
100% РЕШЕНИЕ:
═══════════════════════════════════════════════════════════════════

1. Удалите расширение полностью:
   chrome://extensions/ → Voice-to-Chat → "Удалить"

2. Установите заново:
   Запустите INSTALL.bat из папки

3. При первом открытии:
   → Нажмите зелёную кнопку "🔑 РАЗРЕШИТЬ ДОСТУП"
   → Chrome покажет запрос
   → Нажмите "Разрешить"

4. Готово!


═══════════════════════════════════════════════════════════════════

После перезагрузки расширения всё заработает! 🎉
