# Диаграмма идеального зацикливания 8-секундного видео

## 🔄 Как должно работать зацикливание

```
┌─────────────────────────────────────────────────────────────┐
│                    8-СЕКУНДНОЕ ВИДЕО                         │
│                                                              │
│  0.00 сек                                        8.00 сек   │
│     │                                                │       │
│     ▼                                                ▼       │
│  ┌─────┐                                         ┌─────┐    │
│  │🚛   │ ──────────────────────────────────────▶ │🚛   │    │
│  │     │         Трак движется вперёд            │     │    │
│  └─────┘                                         └─────┘    │
│     ▲                                                │       │
│     │                                                │       │
│     └────────────────────────────────────────────────┘       │
│              ЗАЦИКЛИВАНИЕ (loop back)                        │
│                                                              │
│  ✅ Трак в ОДИНАКОВОЙ позиции на 0.00 и 8.00               │
│  ✅ Камера в ОДИНАКОВОМ положении                           │
│  ✅ Фон СОВПАДАЕТ                                           │
└─────────────────────────────────────────────────────────────┘
```

## ✅ ПРАВИЛЬНОЕ зацикливание

```
Кадр 0.00 сек:          Кадр 8.00 сек:
┌──────────────┐        ┌──────────────┐
│   🌅         │        │   🌅         │  ← Солнце в той же позиции
│              │        │              │
│     🚛       │   =    │     🚛       │  ← Трак в той же позиции
│  ═══════════ │        │  ═══════════ │  ← Дорога совпадает
└──────────────┘        └──────────────┘

Воспроизведение:
0→1→2→3→4→5→6→7→8→0→1→2→3→4→5→6→7→8→0→1→...
                  ↑
            Невидимый стык!
```

## ❌ НЕПРАВИЛЬНОЕ зацикливание

```
Кадр 0.00 сек:          Кадр 8.00 сек:
┌──────────────┐        ┌──────────────┐
│   🌅         │        │      🌅      │  ← Солнце сместилось
│              │        │              │
│     🚛       │   ≠    │       🚛     │  ← Трак в другой позиции!
│  ═══════════ │        │  ═══════════ │  ← Дорога не совпадает
└──────────────┘        └──────────────┘

Воспроизведение:
0→1→2→3→4→5→6→7→8→0→1→2→3→4→5→6→7→8→0→1→...
                  ↑
            ВИДИМЫЙ ПРЫЖОК! ❌
```

## 🎯 Что должно совпадать на 0.00 и 8.00

### 1. Позиция трака
```
0.00 сек:  [🚛────────────────]
8.00 сек:  [🚛────────────────]  ← Та же позиция в кадре
```

### 2. Фаза вращения колёс
```
0.00 сек:  ⚙️ ← Колесо в позиции "12 часов"
8.00 сек:  ⚙️ ← Колесо снова в позиции "12 часов"
```

### 3. Положение камеры
```
0.00 сек:  📹 ← Камера на расстоянии X от трака
8.00 сек:  📹 ← Камера на том же расстоянии X
```

### 4. Фон (дорога, пейзаж)
```
0.00 сек:  [═══|═══|═══|═══]  ← Разметка дороги
8.00 сек:  [═══|═══|═══|═══]  ← Та же разметка
```

## 🔄 Циклическое движение

### Концепция "визуального цикла"

Трак должен пройти такое расстояние за 8 секунд, чтобы визуально 
вернуться в исходное положение:

```
┌─────────────────────────────────────────────────┐
│  Дорога (вид сверху):                           │
│                                                  │
│  ═══════════════════════════════════════════════ │
│  ═══════════════════════════════════════════════ │
│  ═══════════════════════════════════════════════ │
│                                                  │
│  Трак проезжает ровно 1 "визуальный цикл":      │
│                                                  │
│  0 сек: 🚛 (позиция A)                          │
│  ↓                                               │
│  4 сек: ────🚛──── (середина пути)              │
│  ↓                                               │
│  8 сек: 🚛 (позиция A снова!)                   │
│                                                  │
└─────────────────────────────────────────────────┘
```

## 🧪 Тест зацикливания

### Метод "Наложение кадров"

```
Шаг 1: Извлечь первый кадр
┌──────────────┐
│   Кадр 0.00  │
│     🚛       │
└──────────────┘

Шаг 2: Извлечь последний кадр
┌──────────────┐
│   Кадр 8.00  │
│     🚛       │
└──────────────┘

Шаг 3: Наложить с прозрачностью 50%
┌──────────────┐
│  Кадр 0.00   │
│  + Кадр 8.00 │
│     🚛🚛      │  ← Если видно "двоение" — плохо!
└──────────────┘

✅ Идеально: Один чёткий трак (кадры совпадают)
❌ Плохо: Два размытых трака (кадры не совпадают)
```

## 🎯 Финальная проверка

### Чеклист идеального зацикливания:

- [ ] Длительность РОВНО 8.00 секунд
- [ ] Трак в одной позиции на 0:00 и 8:00
- [ ] Камера не меняет скорость
- [ ] Фон однородный (нет уникальных объектов)
- [ ] Освещение постоянное
- [ ] При повторе нет видимого "прыжка"
- [ ] Можно смотреть 10+ циклов без дискомфорта
