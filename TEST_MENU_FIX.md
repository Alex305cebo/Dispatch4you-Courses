# 🧪 Тестирование исправления меню

## Что нужно проверить

### 1. Игра всегда начинается с меню

**Тест 1: Прямое открытие /game**
1. Открой браузер
2. Перейди на `http://localhost:8081/game` (или твой URL)
3. ✅ **Ожидаемый результат:** Автоматическое перенаправление на главное меню `/`

**Тест 2: Открытие через меню**
1. Открой `http://localhost:8081/`
2. Нажми "Начать игру" или "Продолжить"
3. ✅ **Ожидаемый результат:** Игра загружается нормально

**Тест 3: Рефреш страницы в игре**
1. Находясь в игре, нажми F5 (обновить страницу)
2. ✅ **Ожидаемый результат:** 
   - Если есть сохранение — игра восстанавливается
   - Если нет сохранения — перенаправление на меню

### 2. Видео-фон отображается корректно

**Тест 4: Видео на главном меню**
1. Открой главное меню `/`
2. ✅ **Ожидаемый результат:** 
   - Видео с траком проигрывается на фоне
   - Видео замедленное (playbackRate = 0.5)
   - Видео зациклено (loop)
   - Видео затемнено (opacity: 0.6)

**Тест 5: Fallback механизм**
1. Открой DevTools (F12)
2. Перейди на вкладку Network
3. Обнови страницу меню
4. ✅ **Ожидаемый результат:** 
   - Видео загружается с одного из путей
   - Нет ошибок 404 в консоли

### 3. Сохранения работают

**Тест 6: Продолжить игру**
1. Начни новую игру
2. Поиграй немного
3. Вернись на главное меню (кнопка "Выход")
4. Нажми "Продолжить"
5. ✅ **Ожидаемый результат:** Игра загружается с того места где остановился

**Тест 7: Новая игра**
1. Если есть сохранение, нажми "Новая игра"
2. Подтверди предупреждение
3. ✅ **Ожидаемый результат:** Начинается новая игра, старое сохранение удаляется

## Что делать если тесты не проходят

### Проблема: Видео не отображается

**Решение 1:** Проверь что файл существует
```bash
ls -la DispatcherTraining/game/Truck_loop.mp4
```

**Решение 2:** Проверь консоль браузера (F12)
- Ищи ошибки загрузки видео
- Проверь какой путь пытается загрузить

**Решение 3:** Попробуй другой путь в коде
```typescript
// В menu.tsx, строка ~140
src="/game/Truck_loop.mp4"  // Попробуй изменить на:
src="./Truck_loop.mp4"      // или
src="/Truck_loop.mp4"       // или
src="./assets/Truck_loop.mp4"
```

### Проблема: Игра не перенаправляет на меню

**Решение 1:** Очисти sessionStorage
```javascript
// В консоли браузера (F12)
sessionStorage.clear();
location.reload();
```

**Решение 2:** Проверь что изменения применились
```bash
# Проверь файл game.tsx
grep -A 5 "ВСЕГДА перенаправляем" DispatcherTraining/adventure/app/game.tsx
```

**Решение 3:** Перезапусти dev server
```bash
# Ctrl+C чтобы остановить
# Затем снова npm start
```

### Проблема: Сохранения не работают

**Решение 1:** Проверь localStorage
```javascript
// В консоли браузера (F12)
console.log(localStorage.getItem('dispatcher-game-save'));
```

**Решение 2:** Очисти все данные
```javascript
// В консоли браузера (F12)
localStorage.clear();
sessionStorage.clear();
location.reload();
```
