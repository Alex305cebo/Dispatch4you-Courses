# Test Checklist — Profile Integration

## Цель тестирования
Проверить работу нового раздела **Профиль** с 3 слотами сохранений и интеграцией Гаража/Настроек.

---

## ✅ Тест 1: Открытие Профиля

### Шаги:
1. Запустить игру
2. Открыть главное меню
3. Кликнуть на пункт "◉ Профиль"

### Ожидаемый результат:
- Открывается модальное окно ProfilePopup
- Видны 4 вкладки: 💾 Сохранения, 📊 Статистика, 🔧 Гараж, ⚙️ Настройки
- Активна вкладка "Сохранения"
- Видны 3 слота: Слот 1, Слот 2, Слот 3
- Все слоты пустые (если первый запуск)

---

## ✅ Тест 2: Создание новой игры в Слоте 1

### Шаги:
1. Открыть Профиль
2. Кликнуть на "Слот 1" (пустой)
3. Подтвердить "Начать новую игру в Слот 1?"
4. Дождаться загрузки игры

### Ожидаемый результат:
- Игра запускается
- В консоли: `🎮 Starting shift with 1 trucks, session: "Новая смена", slot: 1`
- В консоли: `✅ Game saved to slot 1`
- Игра работает нормально

---

## ✅ Тест 3: Автосохранение в слот

### Шаги:
1. Продолжить игру из Теста 2
2. Подождать 1-2 минуты игрового времени
3. Открыть DevTools → Application → Local Storage
4. Найти ключ `dispatcher-save-slot-1`

### Ожидаемый результат:
- Ключ `dispatcher-save-slot-1` существует
- Содержит JSON с полями: `version`, `phase`, `currentSlotId`, `trucks`, etc.
- `currentSlotId` = 1
- `savedAt` и `lastPlayed` обновляются

---

## ✅ Тест 4: Загрузка из слота

### Шаги:
1. Выйти в главное меню (или обновить страницу)
2. Открыть Профиль
3. Кликнуть на "Слот 1" (теперь заполненный)
4. Подтвердить "Загрузить игру из Слот 1?"

### Ожидаемый результат:
- В консоли: `✅ Loading from slot 1`
- Игра загружается с сохранённого состояния
- Траки на тех же позициях
- Баланс, день, время — как были сохранены

---

## ✅ Тест 5: Отображение данных слота

### Шаги:
1. Создать игру в Слоте 1
2. Поиграть 5-10 минут
3. Выйти в меню
4. Открыть Профиль → Сохранения

### Ожидаемый результат:
- Слот 1 показывает:
  - Иконку 🎮
  - Название сессии: "Новая смена"
  - День и время: "День 1 · только что" (или "X мин назад")
  - Бейджи: 💰 $15,000, 🚛 1, ⭐ 100%
  - Кнопку 🗑️ для удаления

---

## ✅ Тест 6: Несколько слотов параллельно

### Шаги:
1. Создать игру в Слоте 1, поиграть 5 минут
2. Выйти в меню
3. Открыть Профиль → Создать игру в Слоте 2
4. Поиграть 3 минуты
5. Выйти в меню
6. Открыть Профиль → Сохранения

### Ожидаемый результат:
- Слот 1: показывает данные первой игры
- Слот 2: показывает данные второй игры
- Слот 3: пустой
- Можно загрузить любой слот
- Слоты не перезаписывают друг друга

---

## ✅ Тест 7: Удаление слота

### Шаги:
1. Открыть Профиль → Сохранения
2. Кликнуть на 🗑️ возле Слота 1
3. Подтвердить "Удалить сохранение из Слота 1?"

### Ожидаемый результат:
- Слот 1 становится пустым
- Показывает иконку ➕ и текст "Пустой слот — начать новую игру"
- В localStorage ключ `dispatcher-save-slot-1` удалён
- Слоты 2 и 3 не затронуты

---

## ✅ Тест 8: Вкладка Статистика

### Шаги:
1. Создать игру, поиграть 10 минут
2. Открыть Профиль → Статистика

### Ожидаемый результат:
- Видна сетка 3×2 с карточками:
  - 💰 Баланс (зелёный/красный)
  - 📈 Заработано (синий)
  - 📉 Потрачено (оранжевый)
  - ⭐ Репутация (жёлтый)
  - 🚛 Траков (фиолетовый)
  - 📅 День (розовый)
- Значения соответствуют текущей игре

---

## ✅ Тест 9: Вкладка Гараж

### Шаги:
1. Открыть Профиль → Гараж
2. Кликнуть "Открыть гараж"

### Ожидаемый результат:
- ProfilePopup закрывается
- Открывается RepairGarageModal
- Можно отправить трак в гараж, отремонтировать, улучшить

## ✅ Тест 10: Вкладка Настройки

### Шаги:
1. Открыть Профиль → Настройки
2. Кликнуть "Открыть настройки"

### Ожидаемый результат:
- Открывается SettingsPopup поверх ProfilePopup
- Можно изменить настройки графики, звука
- После закрытия SettingsPopup — возвращаемся в ProfilePopup

## ✅ Тест 11: Google Sign-In

### Шаги:
1. Открыть Профиль → Сохранения
2. Прокрутить вниз до карточки "☁️ Облачные сохранения"
3. Кликнуть "Войти через Google"
4. Войти в аккаунт

### Ожидаемый результат:
- Карточка "Облачные сохранения" исчезает
- Вверху ProfilePopup появляется аватар и email
- Внизу появляется кнопка "Выйти из аккаунта"

## ✅ Тест 12: Меню без лишних пунктов

### Шаги:
1. Открыть главное меню

### Ожидаемый результат:
- Видны только 3 пункта:
  - ▶ Продолжить (если есть сохранение)
  - ⚡ Начать игру / Новая игра
  - ◉ Профиль
- НЕТ пунктов: "Сохранения", "Гараж", "Настройки"
- Все функции доступны через Профиль
