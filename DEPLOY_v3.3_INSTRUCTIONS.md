## 📤 Загрузка на Hostinger

### Шаг 1: Вход в панель
1. Открыть https://hpanel.hostinger.com
2. Выбрать сайт **dispatch4you.com**
3. Открыть **File Manager**

### Шаг 2: Создание бэкапа
1. Перейти в `public_html/adventure/`
2. Найти папку `dist`
3. Переименовать: `dist` → `dist.backup.20260502`
4. Убедиться что бэкап создан

### Шаг 3: Загрузка архива
1. Нажать кнопку **Upload** в File Manager
2. Выбрать файл `game-beta-v3.3.0-XXXXXXXX-XXXXXX.zip`
3. Дождаться завершения загрузки (~5-10 минут для 5 MB)

### Шаг 4: Распаковка
1. Кликнуть правой кнопкой на загруженный архив
2. Выбрать **Extract**
3. Указать папку назначения: `dist/`
4. Нажать **Extract**
5. Дождаться завершения

### Шаг 5: Очистка кэша
1. Вернуться в главную панель Hostinger
2. Перейти: **Website** → **Clear Cache**
3. Нажать **Clear Cache**
4. Дождаться подтверждения

## ✅ Проверка после деплоя

### 1. Открыть игру
```
URL: https://dispatch4you.com/game
```

### 2. Проверить версию
- Открыть главное меню
- Найти бейдж **BETA 3.3** (вверху слева)
- Внизу должно быть: **v3.3**

### 3. Проверить кнопку Continue
**Если есть сохранение**:
- ✅ Кнопка "Продолжить" показывается первой
- ✅ Кнопка зелёного цвета с иконкой ▶
- ✅ При нажатии загружается игра

**Если нет сохранения**:
- ✅ Кнопка "Продолжить" НЕ показывается
- ✅ Показывается только "Начать игру"

### 4. Проверить сохранения
1. Начать новую игру
2. Сыграть 1-2 минуты
3. Игра автоматически сохраняется
4. Обновить страницу (F5)
5. Проверить что кнопка Continue появилась
6. Нажать Continue
7. Проверить что игра загрузилась

### 5. Проверить консоль
1. Открыть DevTools (F12)
2. Вкладка **Console**
3. Проверить что нет ошибок (красных сообщений)

## 🐛 Решение проблем

### Проблема 1: Игра не загружается
**Решение**:
1. Очистить кеш браузера (Ctrl+Shift+Delete)
2. Открыть в режиме инкогнито
3. Проверить что файлы распакованы в `dist/`
4. Проверить что `index.html` существует

### Проблема 2: Версия не обновилась
**Решение**:
1. Очистить кеш Hostinger (Website → Clear Cache)
2. Очистить кеш браузера
3. Подождать 5-10 минут
4. Открыть в режиме инкогнито

### Проблема 3: Кнопка Continue не работает
**Решение**:
1. Открыть консоль (F12)
2. Проверить ошибки
3. Проверить что `menu.tsx` обновился
4. Создать новое сохранение и проверить снова

### Проблема 4: Ошибки в консоли
**Решение**:
1. Скопировать текст ошибки
2. Проверить что все файлы загружены
3. Проверить что пути к файлам корректны
4. Откатиться к бэкапу если нужно
