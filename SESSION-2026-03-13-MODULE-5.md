## 📊 ДЕТАЛИ МОДУЛЯ 5

### Структура (6 секторов):

1. **Основы переговоров с брокерами**
   - Типы брокеров (Freight Brokers, 3PL, Shipper Direct, Load Board)
   - Принципы успешных переговоров
   - Структура эффективного звонка (5 этапов)
   - Case Study: Первый звонок новому брокеру
   - Quiz: Контр-оферта при низкой ставке

2. **5 продвинутых техник переговоров**
   - Anchoring (Якорение) - называть цену выше желаемой
   - BATNA - иметь альтернативу
   - Value Stacking - обосновывать высокую цену
   - Silence is Golden - молчание после предложения
   - Package Deal - несколько грузов за лучшую ставку
   - Case Study: Комбинация техник (LA → Phoenix, +$84)
   - Quiz: Использование Silence is Golden

3. **Дополнительные тактики и психология**
   - Mirroring (Зеркалирование)
   - Labeling (Маркировка эмоций)
   - Calibrated Questions (Калиброванные вопросы)
   - 6 принципов влияния: Reciprocity, Scarcity, Authority, Liking
   - Case Study: Психологические техники (+$300)
   - Quiz: Техника Mirroring

4. **Работа с возражениями и сложными ситуациями**
   - Топ-10 возражений брокеров с готовыми ответами
   - Сложные ситуации (агрессивный брокер, снижение ставки)
   - Case Study: Агрессивный брокер (60% успеха)
   - Quiz: Работа с возражением "I have other carriers"

5. **Построение долгосрочных отношений**
   - Преимущества постоянных брокеров (+10-15% к ставкам)
   - 3 этапа построения отношений (1-3, 4-10, 11+ загрузок)
   - Профессиональные практики (CRM, регулярный контакт)
   - Метрики preferred carrier status
   - Case Study: От разовой сделки к партнерству (120+ грузов/год)
   - Quiz: Preferred carrier status

6. **Готовые скрипты для переговоров**
   - 5 телефонных скриптов (первый звонок, постоянный брокер, отказ, запрос условий, агрессивный брокер)
   - Email шаблоны (первый контакт, запрос оплаты, уведомление о задержке, запрос Rate Confirmation)
   - Case Study: Использование скриптов (+14% к ставкам)
   - Quiz: Использование паузы в скрипте
