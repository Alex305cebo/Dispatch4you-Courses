## Troubleshooting

**Порт 3001 занят?**
```bash
netstat -ano | findstr :3001
taskkill /PID <номер> /F
```

**Не загружаются файлы?**
- Проверь консоль браузера (F12)
- Проверь консоль сервера
- Убедись что все файлы на месте

**Игра не работает?**
- Очисти кэш браузера (Ctrl + Shift + Delete)
- Перезапусти сервер (Ctrl + C, потом npm run game:dev)
