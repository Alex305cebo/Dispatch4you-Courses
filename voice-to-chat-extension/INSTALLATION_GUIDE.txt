================================================================================
  VOICE-TO-CHAT CHROME EXTENSION - INSTALLATION GUIDE
  Russian Speech → English Translation in Browser
================================================================================

REQUIREMENTS:
  ✓ Google Chrome, Microsoft Edge, Brave, or any Chromium browser
  ✓ Microphone (for voice input)
  ✓ Internet connection (for translation service)

================================================================================
INSTALLATION STEPS (3 minutes):
================================================================================

STEP 1: EXTRACT FILES
  - Extract all files from the ZIP to any folder on your computer
  - Example: C:\voice-to-chat-extension
  - Keep all files together in one folder!

STEP 2: OPEN CHROME EXTENSIONS
  - Open Google Chrome
  - Type in address bar: chrome://extensions/
  - Press Enter

STEP 3: ENABLE DEVELOPER MODE
  - Look at the TOP RIGHT corner of the page
  - Find toggle switch "Developer mode"
  - Turn it ON (should turn blue)

STEP 4: LOAD EXTENSION
  - Click "Load unpacked" button (appears after enabling Developer mode)
  - Select the folder where you extracted files
  - Click "Select Folder"

STEP 5: DONE!
  - You should see "Voice-to-Chat: RU → EN" in your extensions list
  - Look for the red microphone icon 🔴 in Chrome toolbar (top right)
  - If you don't see it, click the puzzle icon and pin it

================================================================================
HOW TO USE:
================================================================================

METHOD 1: Click Extension Icon
  1. Click the red microphone icon 🔴 in Chrome toolbar
  2. HOLD DOWN the button with your mouse
  3. Speak in RUSSIAN
  4. RELEASE the button
  5. Wait 2-3 seconds → English translation appears
  6. Text is automatically COPIED to clipboard
  7. Press Ctrl+V to paste anywhere!

METHOD 2: Keyboard Shortcut
  1. Press Ctrl+Shift+V (on any webpage)
  2. Extension popup opens
  3. Hold button → Speak → Release
  4. Done!

================================================================================
FIRST TIME SETUP:
================================================================================

GRANT MICROPHONE PERMISSION:
  - When you first use the extension, Chrome will ask for microphone access
  - Click "Allow" in the popup
  - This only happens once

IF PERMISSION PROMPT DOESN'T APPEAR:
  1. Click the lock icon (or info icon) LEFT of the address bar
  2. Find "Microphone" permission
  3. Change to "Allow"
  4. Refresh the page

================================================================================
TROUBLESHOOTING:
================================================================================

PROBLEM: "No microphone access"
SOLUTION:
  - Check Windows microphone settings
  - Go to Settings → Privacy → Microphone → Allow apps to access
  - Restart Chrome

PROBLEM: "Speech not recognized"
SOLUTION:
  - Speak louder and clearer
  - Check your microphone is working (test in Windows Sound settings)
  - Make sure you selected correct microphone device

PROBLEM: Extension doesn't appear in Chrome
SOLUTION:
  - Make sure "Developer mode" is ON in chrome://extensions/
  - Try removing and re-adding the extension
  - Restart Chrome

PROBLEM: Translation not working
SOLUTION:
  - Check internet connection
  - Try again in a few seconds
  - Translation service might be temporarily unavailable

================================================================================
TECHNICAL NOTES:
================================================================================

  - Extension uses Web Speech API (built into Chrome)
  - Translation via Google Translate API
  - All speech recognition happens locally in browser
  - No data is stored or sent to external servers (except translation)
  - Extension ID will be different on each computer (this is normal)

  - Try reinstalling the extension
  - Ensure Chrome is up to date (Settings → About Chrome)
