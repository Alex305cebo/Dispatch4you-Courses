# 🎤 Voice-to-Chat: RU → EN

Chrome расширение для диктовки на русском с автоматическим переводом на английский.

## 📦 Установка (локально)

### Способ 1: Через Chrome

1. Открой Chrome
2. Перейди: `chrome://extensions/`
3. Включи **"Режим разработчика"** (Developer mode) в правом верхнем углу
4. Нажми **"Загрузить распакованное расширение"** (Load unpacked)
5. Выбери папку: `c:\DispatcherTraining\voice-to-chat-extension`
6. Готово! Увидишь иконку 🔴 в панели расширений

### Способ 2: Быстрая установка (BAT файл)

Запусти: `INSTALL_EXTENSION.bat`

---

## 🚀 Использование

### Вариант 1: Через иконку
1. Нажми на иконку расширения 🔴 (справа вверху)
2. **Зажми кнопку мышкой** → Говори на русском → Отпусти
3. Получи английский перевод (автокопирование в буфер)
4. Вставь в чат (Ctrl+V)

### Вариант 2: Горячая клавиша
1. Нажми **Ctrl+Shift+V**
2. Зажми кнопку → Говори → Отпусти
3. Готово!

## 🐛 Решение проблем

### "Нет доступа к микрофону"
1. Нажми на замок/информацию слева от адресной строки
2. Разрешения → Микрофон → Разрешить

### "Речь не распознана"
- Говори громче и чётче
- Проверь микрофон в настройках Windows
- Убедись что выбран правильный микрофон

### Расширение не появляется
1. Перезагрузи Chrome
2. Проверь что "Режим разработчика" включён
3. Переустанови расширение
