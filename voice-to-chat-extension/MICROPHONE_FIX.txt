┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 1: ПЕРВЫЙ ЗАПУСК                                           │
└─────────────────────────────────────────────────────────────────┘

1. Откройте расширение (клик на красную иконку 🔴)

2. Нажмите кнопку "ЗАЖМИ И ГОВОРИ"

3. Chrome покажет ВСПЛЫВАЮЩЕЕ ОКНО вверху:
   
   ┌───────────────────────────────────────────────┐
   │  Voice-to-Chat хочет использовать микрофон    │
   │                                               │
   │        [Блокировать]      [Разрешить]  ←  ✓  │
   └───────────────────────────────────────────────┘
   
4. Нажмите "РАЗРЕШИТЬ" ← ВАЖНО!

5. Готово! Теперь работает


┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 2: ЕСЛИ НЕ ПОЯВИЛОСЬ ВСПЛЫВАЮЩЕЕ ОКНО                      │
└─────────────────────────────────────────────────────────────────┘

1. Посмотрите в АДРЕСНУЮ СТРОКУ Chrome (слева)

2. Найдите иконку микрофона 🎤 или замок 🔒

3. Кликните на неё

4. Найдите "Микрофон" в списке

5. Выберите "Разрешить"

6. Перезагрузите расширение (закройте и откройте снова)


┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 3: ИСПОЛЬЗУЙТЕ ВСТРОЕННУЮ ПОМОЩЬ                           │
└─────────────────────────────────────────────────────────────────┘

В самом расширении есть кнопка помощи!

1. Откройте расширение

2. Нажмите кнопку: "❓ Микрофон не работает?"

3. Откроется окно с:
   ✓ Пошаговыми инструкциями
   ✓ Кнопками быстрого доступа к настройкам
   ✓ Тестом микрофона

4. Нажмите "🎤 Протестировать микрофон"
   → Если работает, увидите "✅ Микрофон работает!"


┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 4: НАСТРОЙКИ CHROME (ЕСЛИ ВСЁ ЕЩЁ НЕ РАБОТАЕТ)            │
└─────────────────────────────────────────────────────────────────┘

СПОСОБ А: Через расширение
   1. Откройте расширение
   2. Нажмите "❓ Микрофон не работает?"
   3. Нажмите "🔧 Открыть настройки микрофона Chrome"
   4. Найдите Voice-to-Chat в списке
   5. Убедитесь что стоит "Разрешить"

СПОСОБ Б: Вручную
   1. Откройте Chrome
   2. Вставьте в адресную строку:
      chrome://settings/content/microphone
   3. Нажмите Enter
   4. Проверьте настройки:
      - "Спрашивать разрешение" → Включено
      - Voice-to-Chat в "Разрешено" → Да


┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 5: НАСТРОЙКИ WINDOWS                                       │
└─────────────────────────────────────────────────────────────────┘

СПОСОБ А: Через расширение
   1. Откройте расширение
   2. Нажмите "❓ Микрофон не работает?"
   3. Нажмите "🪟 Открыть настройки Windows"

СПОСОБ Б: Вручную (Windows 10/11)
   1. Нажмите Win + I (открыть Настройки)
   2. Перейдите: Конфиденциальность → Микрофон
   3. Убедитесь что:
      - "Разрешить приложениям доступ" → ВКЛ
      - Google Chrome в списке → ВКЛ


┌─────────────────────────────────────────────────────────────────┐
│  ШАГ 6: ПРОВЕРКА МИКРОФОНА                                      │
└─────────────────────────────────────────────────────────────────┘

1. Проверьте что микрофон физически подключён

2. Проверьте в Windows:
   - Правый клик на значок звука (трей)
   - "Параметры звука"
   - Раздел "Ввод"
   - Выберите правильный микрофон
   - Говорите → должна двигаться полоска

3. Протестируйте в другой программе:
   - Откройте Диктофон Windows
   - Или Skype/Teams/Zoom
   - Если там работает → проблема в Chrome


┌─────────────────────────────────────────────────────────────────┐
│  БЫСТРЫЕ РЕШЕНИЯ (если ничего не помогло)                       │
└─────────────────────────────────────────────────────────────────┘

1. ПЕРЕЗАГРУЗИТЕ РАСШИРЕНИЕ
   - chrome://extensions/
   - Найдите Voice-to-Chat
   - Нажмите кнопку обновления ⟳

2. ПЕРЕЗАПУСТИТЕ CHROME
   - Закройте все окна Chrome
   - Откройте снова
   - Попробуйте расширение

3. ОБНОВИТЕ CHROME
   - chrome://settings/help
   - Проверьте обновления
   - Установите если есть

4. ПОПРОБУЙТЕ ДРУГОЙ БРАУЗЕР
   - Microsoft Edge
   - Brave
   - Opera
   (все используют тот же движок)

5. ПРОВЕРЬТЕ АНТИВИРУС
   - Может блокировать доступ к микрофону
   - Добавьте Chrome в исключения


┌─────────────────────────────────────────────────────────────────┐
│  ЧАСТЫЕ ОШИБКИ И ИХ РЕШЕНИЯ                                     │
└─────────────────────────────────────────────────────────────────┘

❌ "Нет доступа к микрофону"
   → Нажмите "Разрешить" в Chrome
   → Проверьте chrome://settings/content/microphone

❌ "Микрофон недоступен"
   → Проверьте физическое подключение
   → Закройте другие программы (Skype, Teams)

❌ "Речь не распознана"
   → Говорите громче и чётче
   → Приблизьте микрофон
   → Уберите фоновый шум

❌ Красный крестик на иконке микрофона
   → Клик на иконку → Выбрать "Разрешить"

❌ Нет кнопки "Разрешить"
   → Перезагрузите расширение
   → Попробуйте на другой странице
