📦 SUPER EASY INSTALLATION (2 MINUTES):
═══════════════════════════════════════════════════════════════════


OPTION 1: AUTOMATIC INSTALLER (RECOMMENDED) ⭐
───────────────────────────────────────────────────────────────────

   👉 Double-click:  INSTALL.bat

   It will:
   ✓ Open Chrome Extensions page automatically
   ✓ Open the extension folder
   ✓ Show you exactly what to do

   Then just:
   1. Turn ON "Developer mode" (top right)
   2. Click "Load unpacked"
   3. Select the opened folder
   
   DONE! 🎉


OPTION 2: WEB INSTALLER (EVEN EASIER) 🌟
───────────────────────────────────────────────────────────────────

   👉 Double-click:  QUICK_INSTALL.html

   Follow the 3 steps on the web page:
   1. Click button to open Chrome
   2. Enable Developer mode
   3. Load extension

   That's it!


OPTION 3: MANUAL (IF OPTIONS 1 & 2 DON'T WORK)
───────────────────────────────────────────────────────────────────

   1. Open Chrome
   2. Type:  chrome://extensions/
   3. Turn ON "Developer mode" (top right corner)
   4. Click "Load unpacked"
   5. Select THIS folder (where you see these files)
   6. Done!


═══════════════════════════════════════════════════════════════════
🚀 HOW TO USE AFTER INSTALLATION:
═══════════════════════════════════════════════════════════════════

   1. Look for red microphone icon 🔴 in Chrome toolbar (top right)
   
   2. Click the icon
   
   3. HOLD the button with your mouse
   
   4. SPEAK in Russian
   
   5. RELEASE the button
   
   6. Wait 2 seconds → English translation appears!
   
   7. Translation is automatically COPIED to clipboard
   
   8. Press Ctrl+V to paste anywhere!


💡 KEYBOARD SHORTCUT:
───────────────────────────────────────────────────────────────────

   Press:  Ctrl + Shift + V  (anywhere in Chrome)
   
   Opens Voice-to-Chat instantly!

❓ TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════

   Problem: Can't find the icon in Chrome toolbar
   Solution: Click puzzle icon → Pin "Voice-to-Chat"

   Problem: "No microphone access"
   Solution: Click lock icon in address bar → Allow microphone

   Problem: Speech not recognized
   Solution: Speak louder, check microphone in Windows settings

   Problem: Extension not loading
   Solution: Make sure Developer mode is ON in chrome://extensions/
