# План реализации: Редизайн страницы modules-index.html

## Обзор

Пошаговая переработка `pages/modules-index.html` — от текущей статичной сетки 3×4 к интерактивной геймифицированной странице с компактными карточками 6 в ряд, XP-интеграцией, фильтрацией, прогресс-трекером и микроанимациями. Вся реализация — inline CSS + JS в одном HTML-файле.

## Примечания

- Задачи с `*` — опциональные (property-тесты), можно пропустить для быстрого MVP
- Каждая задача ссылается на конкретные требования для трассируемости
- Контрольные точки обеспечивают инкрементальную валидацию
- Вся реализация в одном файле `pages/modules-index.html` (inline CSS + JS)
- Property-тесты валидируют универсальные свойства корректности из дизайна
