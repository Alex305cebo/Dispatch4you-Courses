# Контекстные табы действий под карточкой трака

## Обзор
Под карточкой трака (в TruckCardOverlay) появляются контекстные кнопки-табы действий, которые зависят от состояния трака. Они отличаются от HUD-табов (Route/Stats/Load/Radio) — это **action-табы** с уникальным стилем.

## Когда появляются

### 1. 📦 "Найти груз" — когда трак `idle`
- Появляется вместо текущего блока "Свободен — ищем груз"
- При нажатии: переключает на вкладку Load Board и прописывает город трака в поиск
- Стиль: голубой акцент (#38bdf8), иконка 📦

### 2. 📞 "Брокер" — когда идут переговоры (negotiation.open && negotiation.load)
- Появляется когда игрок начал переговоры по грузу
- При нажатии: открывает NegotiationChat с текущим грузом
- Стиль: зелёный акцент (#4ade80), иконка 📞, пульсирующий индикатор

### 3. 🚛 "Назначить" — когда есть забуканный груз без трака (bookedLoads)
- Появляется когда груз забукан но не назначен на трак
- При нажатии: показывает список доступных траков для назначения
- Стиль: оранжевый акцент (#fb923c), иконка 🚛

## Дизайн табов

### Стиль — glassmorphism pill-кнопки
```
┌─────────────────────────────────────┐
│  📦 Найти груз из Nashville, TN     │  ← idle трак
└─────────────────────────────────────┘

┌──────────────────┐ ┌────────────────┐
│ 📞 Переговоры    │ │ 🚛 Назначить   │  ← когда оба активны
└──────────────────┘ └────────────────┘
```

- Фон: `rgba(color, 0.12)` с `border: 1.5px solid ${color}55`
- borderRadius: 10
- Padding: 8px 12px
- Анимация появления: fadeIn + slideUp
- При hover: scale(1.02), boxShadow glow
