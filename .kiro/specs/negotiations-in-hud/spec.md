# Переговоры в HUD под карточкой трака

## Цель
Переместить процесс переговоров с брокером из модального окна в HUD-панель под карточкой трака для более удобного UX.

## Текущее поведение
1. Пользователь нажимает "📞 Позвонить брокеру" в LoadBoard
2. Открывается модальное окно NegotiationModal поверх всего экрана
3. После успешных переговоров открывается AssignModal для выбора трака
4. Всё происходит в отдельных модальных окнах

## Желаемое поведение
1. Пользователь нажимает "📞 Позвонить брокеру" в LoadBoard
2. Автоматически выбирается первый свободный трак
3. Карточка трака раскрывается с HUD-панелью
4. В HUD появляется новая вкладка "Negotiation" с чатом переговоров
5. После успешных переговоров груз автоматически назначается на этот трак
6. Всё происходит в одном месте — в HUD под карточкой

## Технические изменения

### 1. TruckCardOverlay.tsx
- Добавить новую вкладку "Negotiation" в TruckHUD
- Показывать NegotiationChat внутри HUD
- Автоматически переключаться на вкладку Negotiation при начале переговоров

### 2. game.tsx
- Изменить `setPendingLoad` чтобы вместо открытия модала:
  - Найти первый свободный трак
  - Выбрать этот трак (selectTruck)
  - Установить activeNegotiation для этого трака
  - Переключиться на вкладку "map" чтобы показать карточку

### 3. gameStore.ts
- Добавить поле `activeNegotiation` в Truck
- Добавить метод `startNegotiationForTruck(truckId, load)`
- Добавить метод `completeNegotiationForTruck(truckId, agreedRate)`

### 4. LoadBoardPanel.tsx
- Изменить onCall чтобы вызывать новую логику

## Преимущества
- ✅ Всё в одном месте — не нужно переключаться между окнами
- ✅ Видно карту и трак во время переговоров
- ✅ Более интуитивный UX
- ✅ Меньше кликов для букинга груза
- ✅ Автоматический выбор трака
