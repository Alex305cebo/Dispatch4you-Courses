## UI UX Pro Max — Использование

### Автоматическая активация
Навык активируется при любой работе с UI/UX. Просто говори естественно:

```
Создай новую страницу для документов
Улучши дизайн карточки трака
Сделай hero-секцию для главной
Какие цвета использовать для финтех-продукта?
Какой шрифт подойдет для дашборда?
```

### Что знает система о DispatcherTraining

**Тип продукта:** Transportation Management System / Dispatcher Training Platform

**UI Стиль:** Soft UI Evolution + Bento Grid
- Мягкие тени
- Плавные переходы (200-300ms)
- Карточная структура

**Цветовая схема:**
```css
--primary: #06b6d4      /* Cyan */
--secondary: #0ea5e9    /* Sky Blue */
--accent: #f59e0b       /* Amber */
--bg: #0f172a           /* Slate 900 */
--text: #e2e8f0         /* Slate 200 */
--text-secondary: #94a3b8
```

**Типографика:** Inter / Work Sans
- Google Fonts: `Inter:wght@400;500;600;700|Work+Sans:wght@400;500;600`

**Ключевые правила:**
- ✅ Весь текст светлый (#e2e8f0 или #fff)
- ✅ Минимум 13px для контента, 12px для лейблов
- ✅ Контраст минимум 4.5:1
- ✅ Плавные hover transitions (150-300ms)
- ❌ Никакого мелкого текста < 13px
- ❌ Никаких emoji как иконки (только SVG)

## Magic MCP — Настройка

### Шаг 1: Получить API ключ
1. Перейди: https://21st.dev/magic/console
2. Создай аккаунт или войди
3. Сгенерируй новый API ключ
4. Скопируй ключ

### Шаг 2: Добавить ключ
1. Открой `.kiro/settings/mcp.json`
2. Найди строку: `"API_KEY": "YOUR_API_KEY_HERE"`
3. Замени `YOUR_API_KEY_HERE` на твой ключ
4. Сохрани файл

### Шаг 3: Перезапуск
1. Перезапусти Kiro
2. Или переподключи MCP сервер через панель MCP Server view

### Проверка
Если всё работает, можешь говорить:
```
Создай современный navigation bar
Сделай glassmorphism карточку с анимацией
Добавь интерактивную кнопку с hover эффектом
```

## Как они работают вместе

```

```

## Troubleshooting

### UI UX Pro Max не работает
1. Проверь файл: `.kiro/steering/ui-ux-pro-max.md`
2. Убедись что `inclusion: auto` в frontmatter
3. Перезапусти Kiro

### Magic MCP не отвечает
1. Проверь API ключ в `.kiro/settings/mcp.json`
2. Убедись что `"disabled": false`
3. Проверь интернет соединение
4. Переподключи MCP сервер
5. Проверь логи: Command Palette → "MCP: Show Logs"

### Конфликт стилей
UI UX Pro Max и Magic MCP дополняют друг друга:
- UI UX Pro Max = правила и рекомендации
- Magic MCP = генерация компонентов
- Конфликтов быть не должно
