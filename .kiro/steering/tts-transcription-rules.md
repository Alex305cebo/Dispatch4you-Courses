# Правила транскрипции для TTS (ElevenLabs)

## Главное правило
Аббревиатуры и английские сокращения писать **побуквенно через пробел** — ElevenLabs произносит каждую букву отдельно.

**Пример:** `США` → `С Ш А`, `TMS` → `Ти Эм Эс`
