# 🔧 Инструкция по очистке сохранения

## Проблема
Вы видите старые значения трака (86%, 76%, 95%) вместо новых (33%, 54%, 33%) потому что игра загружает данные из **localStorage** браузера.

## Решение — Очистка localStorage

### Вариант 1: Через консоль браузера (РЕКОМЕНДУЕТСЯ)

1. Откройте игру: `http://localhost:8082/game`
2. Нажмите **F12** (откроется DevTools)
3. Перейдите на вкладку **Console**
4. Вставьте эту команду и нажмите Enter:

```javascript
localStorage.clear(); location.reload();
```

5. Страница перезагрузится с новыми значениями ✅

### Вариант 2: Через настройки браузера

**Chrome/Edge:**
1. F12 → Application → Storage → Local Storage
2. Найдите `http://localhost:8082`
3. Правый клик → Clear
4. Обновите страницу (F5)

**Firefox:**
1. F12 → Storage → Local Storage
2. Найдите `http://localhost:8082`
3. Правый клик → Delete All
4. Обновите страницу (F5)

### Вариант 3: Через кнопку в игре

В главном меню игры есть кнопка **"Clear Save"** — нажмите её.
